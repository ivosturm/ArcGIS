El GeoAnalytics se centra en l'anàlisi de grans quantitats de dades amb l'èmfasi en els patrons espacials i temporals.

Summarize Data

The Summarize Data toolset contains tools that calculate total counts, lengths, areas, and basic descriptive statistics of features and their attributes within areas or near other features.

  • Aggregate Points calculates statistics about points that fall within specified areas or bins.
  • Build Multi-Variable Grid generates a grid of square or hexagonal bins and calculates values for each bin based on the proximity of each input layer.
  • Describe Dataset calculates a summary of your input, and can optionally generate a sample of your input layer.
  • Join Features calculates statistics about features that share a spatial, temporal, or attribute relationship with other features.
  • Reconstruct Tracks calculates statistics about points or polygons that belong to the same track and reconstructs inputs into tracks.
  • Summarize Attributes calculates statistics about feature or tabular data that share attributes.
  • Summarize Within calculates statistics for area features and attributes that overlap each other.

Consolida punts

Mitjançant una capa d'entitats de punts i una capa d'entitats d'àrees o bins definits per una distància especificada, aquesta eina determina quins punts es troben dins de cada àrea o bin i calcula estadístiques de tots els punts de dins de cada àrea o bin. També podeu aplicar la funció d'interval de temps amb aquesta eina.

A continuació s'indiquen alguns exemples:

  • Donades les ubicacions de punts d'incidents delictius, compteu el nombre de delictes per comtat o per altres districtes administratius.
  • Cerqueu els ingressos més alts i més baixos per ubicacions de franquícies mitjançant bins de 100quilòmetres.

L'eina Crear una quadrícula de diverses variables genera una quadrícula de bins quadrats o hexagonals i calcula els valors de cada bin segons la proximitat d'una o diverses capes d'entrada.

A continuació s'indiquen alguns exemples:

  • Amb diverses capes d'infraestructura de transport públic especificades, quina part de la ciutat és menys accessible amb transport públic?
  • Amb les capes de vies navegables especificades, com ara llacs i rius, com es diu la massa d'aigua més propera a cada ubicació dels EUA?
  • Amb una capa d'ingressos d'una unitat familiar especificada, en quin lloc dels EUA es dona la variació d'ingressos més gran en un radi de 50 milles?


Descriu el dataset

Resumiu les entitats d'entrada amb estadístiques, capes de mostra i visualitzacions. Podeu triar que es mostri una capa de mostra o una capa d'extensió.

A continuació s'indiquen alguns exemples:

  • Consulteu les estadístiques de resum calculades per entendre quins valors conté el vostre dataset.
  • En lloc de dibuixar tot el dataset, creeu una capa de subconjunt per visualitzar les entitats i els seus atributs quan s'afegeixin a un mapa.
  • Estalvieu temps i recursos, i executeu l'anàlisi en una capa de mostra abans d'executar l'anàlisi al dataset sencer.


Uneix les entitats

Podeu utilitzar capes d'entitats o dades tabulars per unir entitats i registres en funció de les relacions especifiques entre les capes o taules d'entrada. Les unions es determinaran per les relacions espacials, temporals i d'atributs i, si ho desitgeu, podeu calcular el resum de les estadístiques.

A continuació s'indiquen alguns exemples:

  • Un cop especificades les ubicacions de punts d'incidents delictius d'una hora, uniu-hi les dades de delinqüència mitjançant l'especificació d'una relació espacial de delictes a una distància d'1 quilòmetre entre si i que s'han produït amb una diferència màxima d'1 hora per determinar si hi ha una seqüència de delictes propers en espai i temps.
  • Amb una taula proporcionada de codis postal amb informació demogràfica i entitats d'àrees que representen edificis residencials, uniu la informació demogràfica a les residències perquè cada residència tingui la informació.


Reconstrueix els recorreguts

Amb una capa amb la funció de temps habilitada d'entitats de punts o polígons que representi un instant en el temps, aquesta eina determina quines entitats d'entrada pertanyen a un recorregut i demana les entrades de manera seqüencial en el temps. Les estadístiques es calculen de manera opcional per a les entitats d'entrada a cada recorregut.

A continuació s'indiquen alguns exemples:

  • Amb les ubicacions de punts i l'hora de les mesures d'un huracà, calculeu la velocitat mitjana i la pressió màxima del vent de l'huracà.
  • Utilitzeu les mesures GPS dels vaixells per reconstruir les rutes marítimes com a línies.


Resumir els atributs

Aquesta eina utilitza dades tabulars o d'entitats per resumir les estadístiques de camps específics.

A continuació s'indiquen alguns exemples:

  • Amb les ubicacions de botigues d'alimentació amb un camp COMPANY_NAME especificades, feu un resum de les botigues pel nom d'empresa amb la finalitat de determinar les estadístiques de cada empresa.
  • Amb la taula de botigues d'alimentació amb els camps COMPANY_NAME i COUNTY proporcionada, feu un resum de les botigues pel nom d'empresa i el comtat amb la finalitat de determinar les estadístiques de cada empresa a cada comtat.


Resumir dins de

Cerca àrees (i porcions d'àrees) que se superposen entre dues capes i calcula estadístiques sobre la superposició.

A continuació s'indiquen alguns exemples:

  • Donada una capa d'àrees de conques hidrogràfiques i una capa d'àrees d'ús del sòl per tipus d'ús, calculeu la superfície total del tipus d'ús del sòl per a cada conca hidrogràfica.
  • Donada una capa de parcel·les d'un comtat i una capa de límits de ciutats, resumiu el valor mitjà de les parcel·les buides en cada ciutat.


Cerca ubicacions

Aquestes eines es fan servir per identificar àrees que compleixin els diversos criteris que especifiqueu.

  • Detecta incidents funciona amb les capes que tenen la funció de temps habilitada per determinar quines entitats compleixen una condició específica.
  • Geocodifica les ubicacions a partir d'una taula converteix adreces en coordenades.
  • Cerca ubicacions similars cerca les ubicacions més similars a una o diverses ubicacions de referència en funció dels criteris que especifiqueu.

Detecta incidents

Aquesta eina funciona amb una capa de punts, línies, àrees o taules que té la funció de temps habilitada i que representa un instant de temps. Mitjançant entitats ordenades de manera seqüencial, anomenades recorreguts, aquesta eina determina quines entitats són incidents d'interès. Els incidents estaran determinats per les condicions que especifiqueu.

A continuació s'indiquen alguns exemples:

  • Detecteu les mesures d'huracans de la categoria 4 segons l'atribut de la velocitat del vent.
  • Amb les mesures GPS del moviment dels vaixells i un camp que representi la velocitat del vaixell, trobeu mesures que superin 1,5 vegades la mitjana de les cinc mesures anteriors.


Geocodifica les ubicacions a partir d'una taula

Converteix adreces en coordenades. Utilitzeu aquesta eina en taules de recursos compartits de fitxers de dades massives.


Cerca ubicacions similars

En funció dels criteris que especifiqueu, per a la cerca ubicacions similars es mesura la similitud de les ubicacions de la capa de cerca de candidats amb una o diverses ubicacions de referència.

A continuació s'indiquen alguns exemples:

  • Per cercar les deu botigues més similars, examineu el nombre d'empleats i les vendes anuals.
  • Per cercar les 100 ciutats més similars, examineu la relació entre la població, el creixement anual i la recaptació d'impostos.

Aquestes eines us ajuden a identificar, quantificar i visualitzar els patrons espacials de les vostres dades.

  • Cerca punts calents identifica clusteritzacions estadísticament significatives al patró espacial de les vostres dades.
  • Cerca clústers de punts cerca clústers d'entitats de punts dins del soroll de l'entorn en funció de la seva distribució espacial.
  • Classificació i regressió forestals crea models i genera prediccions mitjançant una adaptació de l'algoritme de bosc aleatori de Leo Breiman, un mètode d'aprenentatge automàtic supervisat.
  • Regressió lineal generalitzada genera prediccions o modela una variable dependent des del punt de vista de la seva relació amb un conjunt de variables explicatives.

Calcula la densitat

L'eina Calcula la densitat crea un mapa de densitat a partir de les entitats de punts mitjançant la difusió de quantitats conegudes del mateix fenomen (que es representen com a atributs dels punts) al mapa. El resultat és una capa d'àrees que representen la densitat.

A continuació s'indiquen alguns exemples:

  • Càlcul de les densitats d'hospitals en un comtat. La capa de resultats mostrarà les àrees amb una accessibilitat alta i baixa als hospitals. Aquesta informació es pot utilitzar per decidir on s'haurien de construir hospitals nous.
  • Identificació d'àrees amb un alt risc d'incendi forestal en funció de l'historial d'ubicacions d'aquests incendis.
  • Localització de comunitats allunyades de les carreteres principals per planificar on s'haurien de construir carreteres noves.

Cerca punts calents

L'eina Cerca punts calents determinarà si hi ha cap clusterització estadísticament significativa al patró espacial de les vostres dades.

  • Els vostres punts (delictes, arbres, accidents de trànsit) estan realment clusteritzats? Com en podeu estar segur?
  • Heu descobert realment un punt calent estadísticament significatiu (de despesa, mortalitat infantil, notes d'examen sistemàticament altes) o el mapa explicaria una història diferent si en canviéssiu els símbols?
L'eina Cerca punts calents us ajudarà a respondre aquestes preguntes amb confiança.

Manage Data

These tools are used for the day-to-day management of geographic and tabular data.

  • Append Data appends features to an existing hosted feature layer.
  • Calculate Field creates a new field or modifies an existing field and creates a new layer in your contents.
  • Clip Layer copies features that intersect a defined area of interest.
  • Copy to Data Store copies data to your ArcGIS Data Store and creates a layer in your contents.
  • Dissolve Boundaries merges areas that share a common boundary or attribute value.
  • Merge Layers copies all features from two layers into a single result layer.
  • Overlay Layers combines two or more layers into a single layer.
