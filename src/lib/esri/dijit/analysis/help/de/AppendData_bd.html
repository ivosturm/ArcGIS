Daten anhängen

Dieses Werkzeug hängt Daten an einen vorhandenen gehosteten Feature-Layer an. "Daten anhängen" ändert den ursprünglichen Eingabe-Layer und erzeugt keinen neuen Ausgabe-Layer. Sie können Felder basierend auf dem Feldnamen und dem Feldtyp abgleichen oder fortgeschrittene Abgleichmethoden anwenden.

Nachfolgend finden Sie einige Beispiele:

  • Sie sind in einem großen Versicherungsunternehmen angestellt, das monatlich Verkaufsdatensätze erzeugt. Für den Jahresbericht müssen die Monatsdaten zu Analysezwecken zusammengestellt werden. Starten Sie die Verkaufsanalyse mit "Daten anhängen", um den Layer "AnnualReport_2018" einmal pro Monat zu aktualisieren, wenn die monatlichen Zahlen verfügbar sind. Da Ihr Unternehmen bekanntermaßen jeden Monat ein konsistentes Schema beibehält, können Sie die Methode der standardmäßigen Feldzuordnung anwenden, um Features effizient basierend auf Feldnamen und -typ anzuhängen.
  • Zehn Umweltschutzgruppen sammeln jeweils unzählige Klimamessungen innerhalb ihrer Zuständigkeitsbereiche. Die Organisationen möchten ihre Daten zusammenstellen, um regionale und globale Muster besser verstehen zu können. Jede Organisation ermittelt das Schema für ihr Dataset. Sie haben jeweils andere Feldnamen für Temperatur- und Windgeschwindigkeitsdaten angegeben. Aufgrund der unterschiedlichen Feldnamen in den Datasets kann die Feldzuordnung verwendet werden, um anzugeben, wie Temperatur- und Windgeschwindigkeitsmessungen an die jeweiligen Felder angehängt werden sollen.

Layer auswählen an den angehängt werden soll

Die Tabellen-, Punkt-, Linien- und Polygon-Features, an die Daten angehängt werden sollen.

Caution: 

Der Layer an den angehängt wird, wird während der Ausführung des Werkzeugs geändert. Bevor Sie fortfahren, sollten Sie sich vergewissern, dass Sie diesen Layer ändern möchten.

Sie können nicht nur einen Layer aus der Karte auswählen, sondern auch unten in der Dropdown-Liste die Option Analyse-Layer auswählen auswählen, um zu Ihren Inhalten für einen gehosteten Feature-Layer zu navigieren.

Anzuhängenden Layer auswählen

Die Punkt-, Linien- oder Polygon-Features, die an den Eingabe-Layer angehängt werden sollen.

Sie können nicht nur einen Layer aus der Karte auswählen, sondern auch unten in der Dropdown-Liste die Option Analyse-Layer auswählen auswählen, um zu Ihren Inhalten für ein Big-Data-Dateifreigabe-Dataset bzw. -Feature-Layer zu navigieren.

Alle Features anhängen oder eine Teilmenge definieren (optional)

Identifiziert die anzuhängenden Features. Dieser Parameter ist optional.

Festlegen, wie Felder angehängt werden sollen (optional)

Gibt an, wie die Felder des Layers für Anhänge an den Eingabe-Layer angehängt werden. Standardmäßig werden alle übereinstimmenden Eingabefelder und Felder im Anhang angehängt. Felder stimmen überein, wenn sie denselben Feldnamen und denselben Feldtyp aufweisen. Optional können Sie auswählen, wie Felder zugeordnet werden, oder Arcade-Ausdrücke zur Angabe von Feldwerten berechnen.

Sie verfügen beispielsweise über einen Eingabe-Layer mit den Feldern field_number, field_string, date_field und unused_field und einen Layer für Anhänge mit den Feldern field_string, field_number und MY_DATE_FIELD.

  • Standardmäßig ordnet das Werkzeug das Feld field_number im Eingabe-Layer dem Feld field_number im angehängten Layer und das Feld field_string im Eingabe-Layer dem Feld field_string im angehängten Layer zu.
  • Die Felder date_field und unused_field des Eingabe-Layers werden standardmäßig mit dem Wert null gefüllt, da der angehängte Layer kein zugeordnetes Feld für sie enthält.
  • Optional können Sie Werte an Felder anhängen, die nicht automatisch zugeordnet werden. Das Feld date_field im Eingabe-Layer kann beispielsweise dem angehängten Feld MY_DATE_FIELD zugeordnet werden. Verwenden Sie hierzu das Dropdown-Menü neben date_field in der ersten Spalte und wählen Sie im Dropdown-Menü der zweiten Spalte MY_DATE_FIELD aus.
Ein weiteres Beispiel: Ein Eingabe-Layer besitzt die numerischen Felder value_1, value_2, value_3 und my_Average und ein Layer für Anhänge die numerischen Felder value_1, value_2 und value_3.
  • Standardmäßig werden die Felder value_1, value_2 und value_3 zugeordnet und angehangen.
  • Zusätzlich zum Standardverhalten können Werte für das Feld my_Average berechnet werden. Verwenden Sie hierzu mehrere Felder und mathematische Operationen, indem Sie die Dropdown-Option Ausdruck in der Spalte neben my_Average auswählen und einen benutzerdefinierten Arcade-Ausdruck wie 
    average($feature["value_1"] + $feature["value_2"] + $feature["value_3"])
    erstellen.
