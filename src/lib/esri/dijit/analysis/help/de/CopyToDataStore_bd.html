In Data Store kopieren

Dieses Werkzeug verwendet einen Eingabe-Layer und kopiert ihn in einen Data Store. Sie können Ergebnisse in ArcGIS Data Store (Big Data Store vom Typ "relational" oder "spatiotemporal") kopieren, sodass ein Layer in Ihrem Portal erstellt wird, oder Sie können Ergebnisse in eine Big-Data-Datei kopieren.

Sie können beispielsweise Features, die in einer Big-Data-Dateifreigabe gespeichert sind, in einem Data Store vom Typ "relational" speichern und festlegen, dass nur Features innerhalb der aktuellen Kartenausdehnung kopiert werden sollen. Auf diese Weise wird ein gehosteter Feature-Service erstellt, der nur die Features enthält, die sich innerhalb der angegebenen Kartenausdehnung befinden.

Zu kopierenden Layer auswählen

Der Layer oder die Tabelle wird kopiert.

Sie können nicht nur einen Layer aus der Karte auswählen, sondern auch unten in der Dropdown-Liste die Option Analyse-Layer auswählen auswählen, um zu Ihren Inhalten für ein Big-Data-Dateifreigabe-Dataset bzw. -Feature-Layer zu navigieren.

Sie können alle Features verwenden oder eine Auswahl treffen (optional)

Sie können auch die Auswahlschaltfläche verwenden, um die zu kopierenden Positionen zu ermitteln.

Data Store auswählen, in den kopiert werden soll

Der Data Store, in den der Layer kopiert wird. Dieser Parameter überschreibt die Auswahl in den Analyseumgebungen. Folgende Optionen sind verfügbar:

  • ArcGIS Data Store ArcGIS Data Store
    • Spatiotemporal: Dies ist die Standardeinstellung.
    • Relational
  • Ausgabevorlagen für Big-Data-Dateifreigabe
    • Eine Liste der verfügbaren Vorlagen für Big-Data-Dateifreigabe im Format bigDataFileShareName-templateName. Vorlagen für Big-Data-Dateifreigabe müssen vor der Ausführung des Werkzeugs konfiguriert werden.

Name des Ergebnis-Layers

Der Name des Layers, der erstellt wird. Wenn Sie in einen ArcGIS Data Store schreiben, werden Ihre Ergebnisse in Eigene Inhalte gespeichert und der Karte hinzugefügt. Wenn Sie in eine Big-Data-Dateifreigabe schreiben, werden Ihre Ergebnisse in der Big-Data-Dateifreigabe gespeichert und ihrem Manifest hinzugefügt. Sie werden nicht der Karte hinzugefügt. Der Standardname basiert auf dem Werkzeugnamen und dem Namen des Eingabe-Layers. Wenn der Layer bereits vorhanden ist, kann das Werkzeug nicht ausgeführt werden.

Wenn Sie über das Dropdown-Feld Ergebnis speichern in in ArcGIS Data Store (Big Data Store vom Typ "relational" oder "spatiotemporal") schreiben, können Sie den Namen eines Ordners in Eigene Inhalte angeben, in dem das Ergebnis gespeichert wird.
