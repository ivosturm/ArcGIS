Dataset beschreiben

Das Werkzeug "Dataset beschreiben" bietet einen detaillierteren Einblick in Ihr Dataset. Ermitteln Sie mit diesem Werkzeug, worum es sich bei Ihren Eingabedaten handelt und wo sie sich befinden, bevor Sie Analysen für große Datenmengen durchführen. Das Ergebnis ist standardmäßig eine JSON-Zeichenfolge, die wichtige Deskriptoren Ihrer Eingabe-Layer darstellen, und eine Tabelle mit Details zur Summenstatistik für jedes Feld.

Sie können auch zusätzliche Ausgabe-Feature-Layer erstellen, um Ihre Daten weiter zu beschreiben. Zu den Optionen zählen ein Beispiel-Layer oder ein Ausdehnungs-Layer. Sie können einen oder beide dieser Layer als Ausgabe festlegen.

Angenommen, Sie erhalten eine Big-Data-Dateifreigabe mit 15 Datasets. Jedes Dataset verfügt über 10 Millionen Flächen-Features, die Gebäude und Häuser in verschiedenen Regionen Ihres Landes darstellen. Ihre Aufgabe besteht darin, Region C zu analysieren, Ihnen ist jedoch nicht bekannt, in welchem Dataset diese Features enthalten sind. Um zu ermitteln, welches Dataset Sie verwenden sollten, führen Sie das Werkzeug "Dataset beschreiben" aus, und erstellen Sie einen Ausdehnungs-Layer, um zu untersuchen, welches Dataset sich in Ihrer Region befindet.

Ein weiteres Beispiel: Stellen Sie sich vor, Sie wurden mit der Aufgabe betraut, einen Analyse-Workflow für eine große Datenmenge auszuführen. Sie möchten den Workflow testen, aber dieser Vorgang könnte für das vollständige Dataset Stunden oder Tage in Anspruch nehmen. Erstellen Sie einen Beispiel-Layer, an dem Sie den vollständigen Workflow effizient testen können, anstatt Zeit und Ressourcen für die Ausführung der vollständigen Analyse in Anspruch zu nehmen.

Zu beschreibendes Dataset auswählen

Der Layer mit Punkt-, Linien-, Flächen- oder tabellarischen Features, der beschrieben, zusammengefasst und als Stichprobe verwendet wird.

Sie können nicht nur einen Layer aus der Karte auswählen, sondern auch unten in der Dropdown-Liste die Option Analyse-Layer auswählen auswählen, um zu Ihren Inhalten für ein Big-Data-Dateifreigabe-Dataset bzw. -Feature-Layer zu navigieren.

Erstellen Sie zusätzliche beschreibende Layer als Ausgabe, um ein besseres Verständnis Ihrer Big Data zu erlangen. Sie können keinen, einen oder beide der folgenden Layer als Ausgabe festlegen:

  • Ausdehnungs-Layer: Erstellt ein Flächen-Feature, das die Ausdehnung Ihrer Eingabe-Features bzw. Ihres Eingabe-Interessenbereichs darstellt.
  • Beispiel-Layer: Erstellt einen Teilmengen-Layer mit einer bestimmten Menge von Eingabe-Features über Ihr Dataset oder Ihren Interessenbereich.

Beispiel-Layer

Erstellen Sie einen Layer mit einer Teilmenge der Features aus Ihrem Eingabe-Layer als Ausgabe. Wenn Beispiel-Layer ausgewählt ist, können Sie die Anzahl der Features festlegen, die im Beispiel-Layer zurückgegeben werden sollen. Im Ausgabe-Layer werden standardmäßig 100 Beispiel-Features zurückgegeben.

Der Wert muss größer 0 sein. Wenn Sie eine größere Anzahl als die Gesamtzahl der Features festlegen, werden alle Features zurückgegeben.

Bei Auswahl von Aktuelle Kartenausdehnung verwenden enthält der Beispiel-Layer Features, die sich in der Ausdehnung der Karte befinden.

Ausdehnungs-Layer

Erstellen Sie einen Feature-Layer mit einem einzelnen Feature, das die Ausdehnung der Eingabe-Features darstellt, als Ausgabe, indem Sie die Schaltfläche Ausdehnungs-Layer auswählen. Diese Option ist standardmäßig ausgewählt.

Bei Auswahl von Aktuelle Kartenausdehnung verwenden stellt der Ausdehnungs-Layer die sichtbare Ausdehnung der Karte dar.

Name des Ergebnis-Layers

Der Name des Layers, der erstellt wird. Wenn Sie in einen ArcGIS Data Store schreiben, werden Ihre Ergebnisse in Eigene Inhalte gespeichert und der Karte hinzugefügt. Wenn Sie in eine Big-Data-Dateifreigabe schreiben, werden Ihre Ergebnisse in der Big-Data-Dateifreigabe gespeichert und ihrem Manifest hinzugefügt. Sie werden nicht der Karte hinzugefügt. Der Standardname basiert auf dem Werkzeugnamen und dem Namen des Eingabe-Layers. Wenn der Layer bereits vorhanden ist, kann das Werkzeug nicht ausgeführt werden.
