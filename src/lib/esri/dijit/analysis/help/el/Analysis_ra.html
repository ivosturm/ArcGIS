Η ανάλυση raster σάς επιτρέπει να εκτελείτε ανάλυση μεγάλων συνόλων raster δεδομένων χρησιμοποιώντας το ArcGIS Image Server. Έτσι μπορείτε να αναλύετε γρηγορότερα περισσότερα δεδομένα αξιοποιώντας την ισχύ του server. Επί του παρόντος μέσω της διεπαφής web του χρήστη στο Portal for ArcGIS είναι διαθέσιμα τα σύνολα εργαλείων Σύνοψη Δεδομένων, Ανάλυση Μοτίβων, Χρήση Γειτνίασης, Ανάλυση Εικόνας, Ανάλυση Εδάφους, Διαχείριση Δεδομένων και Βαθιά Μάθηση.

Σύνοψη Δεδομένων

Αυτό το σύνολο εργαλείων περιλαμβάνει ένα εργαλείο για τον υπολογισμό στατιστικών στοιχείων για ένα raster θεματικό επίπεδο εντός των ορίων των περιοχών που ορίζετε.

  • Το εργαλείο Σύνοψη Raster εντός υπολογίζει τα στατιστικά στοιχεία για τα κελιά του raster που βρίσκονται εντός των ορίων καθορισμένων περιοχών.

Σύνοψη Raster εντός

Υπολογίζει μερικά συνοπτικά στατιστικά στοιχεία για τα κελιά του raster που βρίσκονται εντός των καθορισμένων περιοχών.

Μερικές ενδεικτικές εφαρμογές είναι οι εξής:

  • Με δεδομένο ένα θεματικό επίπεδο περιοχών λεκάνης απορροής και ένα raster θεματικό επίπεδο υψομέτρου, υπολογίστε το εύρος των υψομέτρων εντός κάθε λεκάνης απορροής για να προσδιορίσετε ποια θα έχουν την υψηλότερη ροή ενέργειας.
  • Με δεδομένο ένα θεματικό επίπεδο διοικητικών μονάδων διαχείρισης και ένα raster θεματικό επίπεδο κατηγοριών χρήσης γης, υπολογίστε την ποικιλία των τιμών χρήσης γης εντός κάθε περιοχής διαχείρισης για να προσδιορίσετε ποιες απαιτούν την περισσότερη δουλειά για την κατάρτιση διαχειριστικού πλάνου.


Ανάλυση Mοτίβων

Αυτά τα εργαλεία σάς βοηθούν να προσδιορίσετε, να προσδιορίσετε ποσοτικά και να απεικονίσετε χωρικά μοτίβα στα δεδομένα σας.

  • Το εργαλείο Υπολογισμός Πυκνότητας παίρνει γνωστές ποσότητες κάποιου φαινομένου και δημιουργεί ένα χάρτη πυκνότητας, διασπείροντας αυτές τις ποσότητες σε όλον το χάρτη.
  • Το εργαλείο Παρεμβολή Σημείων προβλέπει τιμές σε νέες τοποθεσίες, με βάση τις μετρήσεις που βρίσκονται σε μια συλλογή σημείων.

Υπολογισμός Πυκνότητας

Το εργαλείο Ανάλυση Πυκνότητας παίρνει γνωστές ποσότητες κάποιου φαινομένου και δημιουργεί ένα χάρτη πυκνότητας, διασπείροντας αυτές τις ποσότητες σε όλον το χάρτη. Μπορείτε να χρησιμοποιήσετε αυτό το εργαλείο, για παράδειγμα, για να εμφανίσετε συγκεντρώσεις χτυπημάτων κεραυνών ή ανεμοστρόβιλων, πρόσβαση σε νοσοκομεία και πυκνότητες πληθυσμών.


Παρεμβολή Σημείων

Αυτό το εργαλείο σάς επιτρέπει να προβλέψετε τιμές σε νέες τοποθεσίες, με βάση τις μετρήσεις από μια συλλογή σημείων. Το εργαλείο παίρνει δεδομένα σημείου με τιμές σε κάθε σημείο και επιστρέφει περιοχές ταξινομημένες με προβλεπόμενες τιμές. Μπορείτε να χρησιμοποιήσετε αυτό το εργαλείο, για παράδειγμα, για να προβλέψετε τα επίπεδα βροχόπτωσης σε μια λεκάνη απορροής, με βάση τις μετρήσεις που ελήφθησαν από μεμονωμένες μετρήσεις βροχής.


Χρήση Εγγύτητας

Αυτά τα εργαλεία σάς βοηθούν να απαντήσετε σε μία από τις πιο κοινές ερωτήσεις στη χωρική ανάλυση: "Τι υπάρχει κοντά σε τι;"

  • Το εργαλείο Υπολογισμός Απόστασης υπολογίζει την ευκλείδεια απόσταση, την κατεύθυνση και την εκχώρηση από μία αφετηρία ή από ένα σύνολο αφετηριών.
  • Το εργαλείο Καθορισμός Δικτύου Βέλτιστου Κόστους Μετακίνησης υπολογίζει το δίκτυο βέλτιστου κόστους από ένα σύνολο περιοχών εισόδου.
  • Το εργαλείο Καθορισμός Διαδρομής Κόστους Μετακίνησης Ως Πολύγραμμου υπολογίζει την πολύγραμμη διαδρομή ελάχιστου κόστους μεταξύ αφετηριών και γνωστών προορισμών.

Υπολογισμός απόστασης

Υπολογίζει την ευκλείδεια απόσταση, την κατεύθυνση και την εκχώρηση από μία αφετηρία ή από ένα σύνολο αφετηριών. Μπορείτε να χρησιμοποιήσετε το εργαλείο αυτό για να προσδιορίσετε πόσο απέχει μια τοποθεσία από έναν δρόμο, ένα κτήριο ή ένα πάρκο. Μπορείτε επίσης να προσδιορίσετε σε ποια κατεύθυνση πρέπει να μετακινηθείτε από μια τοποθεσία για επιστροφή σε μια αφετηρία ακολουθώντας την πιο ευθεία οδό. Μπορείτε να δείτε για κάθε τοποθεσία στην περιοχή μελέτης σας ποια είναι η πλησιέστερη αφετηρία.


Καθορισμός Δικτύου Βέλτιστου Κόστους Μετακίνησης

Υπολογίζει το δίκτυο βέλτιστου κόστους από ένα σύνολο περιοχών εισόδου.


Καθορισμός Διαδρομής Κόστους Μετακίνησης Ως Σύνθετης Γραμμής

Υπολογίζει την πολύγραμμη διαδρομή ελάχιστου κόστους μεταξύ αφετηριών και γνωστών προορισμών.


Ανάλυση Εικόνας

Τα εργαλεία που ακολουθούν σας βοηθούν στην ανάλυση εικόνων.

  • Το εργαλείο Εφαρμογή Raster Function Template επεξεργάζεται τις δορυφορικές εικόνες σας με την αλυσίδα συναρτήσεων που καθορίζεται από το raster function template.
  • Το εργαλείο Καταγραφή Βλάστησης εντοπίζει τη βλάστηση σε μια εικόνα εκτελώντας μαθηματικές πράξεις στις φασματικές ζώνες.

Εφαρμογή Προτύπου Συναρτήσεων Raster

Επεξεργάζεται τις δορυφορικές εικόνες σας με την αλυσίδα συναρτήσεων που καθορίζεται από το raster function template.


Καταγραφή βλάστησης

Εφαρμόζει ένα αριθμητικό μοντέλο στις ζώνες ενός πολυφασματικού raster θεματικού επιπέδου για να αποκαλύψει πληροφορίες για την κάλυψη βλάστησης της περιοχής μελέτης.


Ανάλυση εδάφους

Αυτά τα εργαλεία σάς βοηθούν να αναλύσετε επιφάνειες raster.

  • Το εργαλείο Υπολογισμός Κλίσης υπολογίζει την κλίση του ανάγλυφου.
  • Το εργαλείο Παραγωγή Προοπτικής υπολογίζει την κατεύθυνση της πιο απότομης κλίσης για κάθε τοποθεσία.
  • Το εργαλείο Δημιουργία Ανάλυσης Ορατότητας προσδιορίζει ό,τι είναι ορατό από οποιοδήποτε σημείο του τοπίου.
  • Το εργαλείο Λεκάνη Απορροής υπολογίζει τη συμβάλλουσα περιοχή επάνω από την εντοπισμένη θέση (ή τις εντοπισμένες θέσεις) σε μια επιφάνεια.

Υπολογισμός κλίσης

Προσδιορίζει μια επιφάνεια που δείχνει την κλίση των δεδομένων υψομέτρου εισόδου. Η κλίση αντιπροσωπεύει το ρυθμό αλλαγής του υψομέτρου για κάθε κελί ψηφιακού μοντέλου εδάφους (DEM).


Παραγωγή προοπτικής

Προσδιορίζει την καθοδική κατεύθυνση του μέγιστου ρυθμού αλλαγής της τιμής από κάθε κελί στα γειτονικά του. Η προοπτική μπορεί να θεωρηθεί ως η κατεύθυνση της κλίσης.


Δημιουργία Ανάλυσης Ορατότητας

Προσδιορίζει τις τοποθεσίες μιας επιφάνειας raster που είναι ορατές σε μια ομάδα παρατηρητών.


Λεκάνες Απορροής

Καθορίζει τη συμβάλλουσα περιοχή επάνω από ένα σύνολο κελιών σε ένα raster.


Διαχείριση δεδομένων

Αυτά τα εργαλεία χρησιμοποιούνται τόσο για την καθημερινή διαχείριση γεωγραφικών δεδομένων όσο και για το συνδυασμό δεδομένων πριν από την ανάλυση.

  • Το εργαλείο Εξαγωγή raster εκτελεί αποκοπή τμημάτων από ένα raster σύνολο δεδομένων για περαιτέρω ανάλυση.
  • Το εργαλείο Νέα αποτύπωση τιμών αντιστοιχίζει διαφορετικές τιμές σε κελιά του raster.
  • Το εργαλείο Μετατροπή στοιχείου σε raster εφαρμόζει ένα raster σε διανυσματικά δεδομένα.
  • Το εργαλείο Μετατροπή raster σε στοιχείο μετατρέπει raster δεδομένα σε διανυσματικά.

Εξαγωγή raster

Εξαγάγετε κελιά από ένα raster με βάση μια τιμή, ένα σχήμα ή την έκταση ενός διαφορετικού συνόλου δεδομένων.


Νέα αποτύπωση τιμών

Αλλάξτε μεμονωμένες τιμές κελιών ή εύρη τιμών κελιών σε νέες τιμές.


Μετατροπή στοιχείου σε raster

Δημιουργήστε ένα νέο raster σύνολο δεδομένων από ένα υπάρχον σύνολο δεδομένων στοιχείων.


Μετατροπή raster σε στοιχείο

Δημιουργήστε ένα νέο σύνολο δεδομένων στοιχείων από ένα υπάρχον raster σύνολο δεδομένων.


Βαθιά μάθηση συνόλων εργαλείων

Τα εργαλεία αυτά χρησιμοποιούνται για την ανίχνευση συγκεκριμένων στοιχείων σε μια εικόνα ή για την ταξινόμηση των pixel σε ένα raster dataset. Η βαθιά μάθηση είναι ένας τύπος μεθόδου μάθησης μέσω μιας μηχανής τεχνητής νοημοσύνης η οποία ανιχνεύει στοιχεία σε δορυφορικές εικόνες που χρησιμοποιούν πολλαπλά θεματικά επίπεδα σε νευρωνικά δίκτυα όπου το κάθε θεματικό επίπεδο έχει τη δυνατότητα εξαγωγής ενός ή περισσότερων μοναδικών στοιχείων στην εικόνα. Τα εργαλεία αυτά χρησιμοποιούν τα μοντέλα που έχουν εκπαιδευτεί για να ανιχνεύουν συγκεκριμένα στοιχεία σε πλαίσια βαθιάς μάθησης τρίτων εταιρειών —όπως τα TensorFlow, CNTK και Keras—και να εξάγουν στοιχεία ή χάρτες με κατηγορίες.

  • Το εργαλείο Ταξινόμηση Pixel Μέσω Βαθιάς Μάθησης παράγει ένα ταξινομημένο raster χρησιμοποιώντας ένα εκπαιδευμένο μοντέλο βαθιάς μάθησης.
  • Το εργαλείο Ανίχνευση Αντικειμένων Μέσω Βαθιάς Μάθησης παράγει ένα feature class που περιέχει τα αντικείμενα τα οποία βρίσκει σε ένα raster εισόδου χρησιμοποιώντας ένα εκπαιδευμένο μοντέλο βαθιάς μάθησης.

Ταξινόμηση Pixel Μέσω Βαθιάς Μάθησης

Εκτελεί ένα εκπαιδευμένο μοντέλο βαθιάς μάθησης σε ένα raster εισόδου για να παραχθεί ένα ταξινομημένο raster στο οποίο κάθε έγκυρο pixel έχει μια ετικέτα κλάσης εκχωρημένη.


Ανίχνευση Αντικειμένων Μέσω Βαθιάς Μάθησης

Εκτελεί ένα εκπαιδευμένο μοντέλο βαθιάς μάθησης σε ένα raster εισόδου για να παραχθεί ένα feature class που θα περιέχει τα αντικείμενα τα οποία βρίσκει. Τα στοιχεία μπορούν να είναι πλαίσια ή πολύγωνα οριοθέτησης γύρω από τα αντικείμενα που έχουν βρεθεί ή σημεία στα κέντρα των αντικειμένων.
