Εμπλουτισμός Από Κάνναβο Πολλαπλών Μεταβλητών

Το εργαλείο Εμπλουτισμός Από Κάνναβο Πολλαπλών Μεταβλητών συνδέει αποτελεσματικά τα γνωρίσματα από ένα θεματικό επίπεδο καννάβου πολλαπλών μεταβλητών σε ένα θεματικό επίπεδο σημείων.

Κάνναβος πολλαπλών μεταβλητών είναι οποιοδήποτε θεματικό επίπεδο που έχει δημιουργηθεί με το εργαλείο «Δημιουργία Καννάβου Πολλαπλών Μεταβλητών» και συγκεντρώνει γνωρίσματα από πολλαπλά θεματικά επίπεδο σε έναν ενιαίο κάνναβο πολυγώνων. Με τη χρήση ενός καννάβου πολλαπλών μεταβλητών για εμπλουτισμό έχετε τη δυνατότητα να προσθέτετε γρήγορα μεγάλες και ποικίλες συλλογές πληροφοριών σε δεδομένα σημείων για χρήση σε περαιτέρω χωρική ανάλυση.

Για παράδειγμα, ας υποθέσουμε ότι έχετε αποκτήσει ένα θεματικό επίπεδο νέων εγκληματικών συμβάντων που ανέκυψαν την προηγούμενη χρονιά σε μια πόλη. Θέλετε να μοντελοποιήσετε τις σχέσεις μεταξύ της συχνότητας των εγκλημάτων και ποικίλων δημογραφικών, οικονομικών και γεωγραφικών παραγόντων. Έχετε δικαίωμα πρόσβασης σε δεδομένα που περιγράφουν τους παράγοντες αυτούς, αλλά είναι αποθηκευμένα σε πέντε διαφορετικά θεματικά επίπεδα με διαφορετικούς τύπους γεωμετρίας. Συν τοις άλλοις, ορισμένα θεματικά επίπεδα είναι αποθηκευμένα σε ένα κοινόχρηστο αρχείο big data ενώ άλλα είναι αποθηκευμένα ως διάφορα feature service. Δημιουργώντας έναν κάνναβο πολλαπλών μεταβλητών για να αποθηκεύσετε τις πληροφορίες αυτές και ύστερα συνδέοντας τον κάνναβο στα εγκληματικά συμβάντα σας μέσω του εργαλείου Εμπλουτισμός Από Κάνναβο Πολλαπλών Μεταβλητών, έχετε τη δυνατότητα να συνδέσετε αποτελεσματικά ένα ευρύ φάσμα γνωρισμάτων στο θεματικό σας επίπεδο σημείων χρησιμοποιώντας μόνο δύο φορές το ίδιο εργαλείο. Ύστερα μπορείτε να χρησιμοποιήσετε τα συνδεδεμένα γνωρίσματα ως επεξηγηματικές μεταβλητές και να αρχίσετε να δημιουργείτε ένα μοντέλο συχνότητας εγκλημάτων με το εξαγόμενο θεματικό επίπεδο.

Επιλογή των σημειακών στοιχείων προς εμπλουτισμό

Τα σημειακά στοιχεία που θα εμπλουτιστούν από τον κάνναβο πολλαπλών μεταβλητών.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να χρησιμοποιήσετε την Επιλογή θεματικού επιπέδου ανάλυσης στο κάτω μέρος της αναπτυσσόμενης λίστας για περιήγηση στο περιεχόμενό σας και αναζήτηση ενός κοινόχρηστου συνόλου δεδομένων αρχείων big data ή ενός feature layer.

Επιλογή του καννάβου πολλαπλών μεταβλητών από τον οποίο θα γίνει ο εμπλουτισμός

Το θεματικό επίπεδο του καννάβου πολλαπλών μεταβλητών το οποίο περιέχει γνωρίσματα που θα συνδεθούν στα σημειακά στοιχεία εισόδου. Το θεματικό επίπεδο πρέπει να έχει δημιουργηθεί μέσω του εργαλείου «Δημιουργία Καννάβου Πολλαπλών Μεταβλητών» και πρέπει να είναι hosted feature layer. Τα θεματικά επίπεδα του καννάβου πολλαπλών μεταβλητών τα οποία είναι αποθηκευμένα σε ένα κοινόχρηστο αρχείο big data δεν υποστηρίζονται ως δεδομένα εισόδου στο εργαλείο αυτό.

Επιλογή των μεταβλητών που θα προστεθούν

Τα γνωρίσματα στο θεματικό επίπεδο του καννάβου πολλαπλών μεταβλητών τα οποία θα συνδεθούν στα σημειακά στοιχεία εισόδου κάθε φορά που τα σημειακά στοιχεία τέμνουν τον κάνναβο.

Όνομα εξαγόμενου θεματικού επιπέδου

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί. Αν πραγματοποιείτε εγγραφή σε ένα ArcGIS Data Store, τα αποτελέσματά σας θα αποθηκευτούν στον φάκελο Το περιεχόμενό μου και θα προστεθούν στον χάρτη. Αν πραγματοποιείτε εγγραφή σε ένα κοινόχρηστο αρχείο big data, τα αποτελέσματά σας θα αποθηκευτούν στο κοινόχρηστο αρχείο big data και θα προστεθούν στη διακήρυξή του. Δεν θα προστεθούν στον χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.
