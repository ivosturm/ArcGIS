Εύρεση Συστάδων Σημείων

Το εργαλείο Εύρεση Συστάδων Σημείων βρίσκει συστάδες στοιχείων σημείων εντός του περιβάλλοντος θορύβου με βάση τη χωρική κατανομή τους.

Για παράδειγμα, ένας μη κυβερνητικός οργανισμός μελετά μια συγκεκριμένη ασθένεια που μεταδίδεται μέσω παρασίτων. Έχει ένα σύνολο δεδομένων σημείων που αναπαριστούν νοικοκυριά σε μια περιοχή μελέτης, ορισμένα από τα οποία είναι μολυσμένα και ορισμένα άλλα όχι. Ένας αναλυτής μπορεί να χρησιμοποιήσει το εργαλείο Εύρεση Συστάδων Σημείων για να εντοπίσει συστάδες μολυσμένων νοικοκυριών που θα βοηθήσουν στην οριοθέτηση μιας περιοχής στην οποία θα ξεκινήσει η θεραπεία και η εξολόθρευση των παρασίτων.

Επιλέξτε το θεματικό επίπεδο για το οποίο θα βρεθούν συστάδες.

Το θεματικό επίπεδο σημείων στο οποίο θα βρεθούν οι συστάδες. Τα θεματικά επίπεδα πρέπει να βρίσκονται σε μια προβολική χωρική αναφορά ή να έχουν μια χωρική αναφορά επεξεργασίας καθορισμένη σε ένα προβολικό σύστημα συντεταγμένων που χρησιμοποιεί την επιλογή Περιβάλλοντα Ανάλυσης.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να χρησιμοποιήσετε την Επιλογή θεματικού επιπέδου ανάλυσης στο κάτω μέρος της αναπτυσσόμενης λίστας για περιήγηση στο περιεχόμενό σας και αναζήτηση ενός κοινόχρηστου συνόλου δεδομένων αρχείων big data ή ενός feature layer.

Επιλογή μεθόδου δημιουργίας συστάδων προς χρήση

Η μέθοδος δημιουργίας συστάδων που θα χρησιμοποιηθεί για τη διάκριση συστάδων σημειακών στοιχείων από τον περιβάλλοντα θόρυβο. Μπορείτε να επιλέξετε να χρησιμοποιήσετε μια καθορισμένη απόσταση ή έναν αυτορυθμιζόμενο αλγόριθμο δημιουργίας συστάδων.

Η μέθοδος καθορισμένης απόστασης (DBSCAN) χρησιμοποιεί ένα καθορισμένο εύρος αναζήτησης για τον διαχωρισμό πυκνών συστάδων από αραιότερο θόρυβο. Η μέθοδος καθορισμένης απόστασης (DBSCAN) είναι ταχύτερη μέθοδος αλλά ενδείκνυται μόνο αν υπάρχει διαθέσιμο για χρήση ένα πολύ σαφές εύρος αναζήτησης το οποίο είναι αποτελεσματικό για τον καθορισμό όλων των συστάδων που ενδέχεται να υπάρχουν. Η μέθοδος καθορισμένης απόστασης (DBSCAN) βρίσκει συστάδες που έχουν παρόμοιες πυκνότητες.

Η μέθοδος αυτορύθμισης (HDBSCAN) δεν απαιτεί τον καθορισμό εύρους αναζήτησης αλλά είναι πιο χρονοβόρα. Η μέθοδος αυτορύθμισης (HDBSCAN) βρίσκει συστάδες σημείων με παρόμοιο τρόπο όπως η μέθοδος καθορισμένης απόστασης (DBSCAN) αλλά χρησιμοποιεί ποικίλα εύρη αναζήτησης που επιτρέπουν την ύπαρξη συστάδων με ποικίλες πυκνότητες.

Ελάχιστος αριθμός σημείων που μπορεί να θεωρηθεί συστάδα

Η παράμετρος αυτή χρησιμοποιείται διαφορετικά ανάλογα με την επιλεγμένη μέθοδο δημιουργίας συστάδων:

  • Μέθοδος καθορισμένης απόστασης (DBSCAN)—Καθορίζει τον αριθμό των στοιχείων που πρέπει να βρεθούν εντός μιας συγκεκριμένης απόστασης από ένα σημείο ώστε το σημείο αυτό να αρχίσει να σχηματίζει μια συστάδα. Η απόσταση καθορίζεται μέσω της μεταβλητής Περιορισμός του εύρους αναζήτησης σε.
  • Μέθοδος αυτορύθμισης (HDBSCAN)—Καθορίζει τον αριθμό των στοιχείων που γειτονεύουν με κάθε σημείο (συμπεριλαμβανομένου του κάθε σημείου) και θα ληφθούν υπόψη κατά τον υπολογισμό της πυκνότητας. Ο αριθμός αυτός είναι επίσης το ελάχιστο μέγεθος κάθε συστάδας το οποίο θα επιτραπεί κατά την εξαγωγή συστάδων.

Περιορισμός του εύρους αναζήτησης σε

Κατά τη χρήση της μεθόδου καθορισμένης απόστασης (DBSCAN), η παράμετρος αυτή είναι η απόσταση εντός της οποίας πρέπει να βρεθεί ο Ελάχιστος αριθμός σημείων που μπορεί να θεωρηθεί συστάδα. Η παράμετρος αυτή δεν χρησιμοποιείται όποτε επιλέγεται η μέθοδος αυτορύθμισης (HDBSCAN) ως η μέθοδος που θα χρησιμοποιηθεί για τη δημιουργία συστάδων.

Όνομα εξαγόμενου θεματικού επιπέδου

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί. Αν πραγματοποιείτε εγγραφή σε ένα ArcGIS Data Store, τα αποτελέσματά σας θα αποθηκευτούν στον φάκελο Το περιεχόμενό μου και θα προστεθούν στον χάρτη. Αν πραγματοποιείτε εγγραφή σε ένα κοινόχρηστο αρχείο big data, τα αποτελέσματά σας θα αποθηκευτούν στο κοινόχρηστο αρχείο big data και θα προστεθούν στη διακήρυξή του. Δεν θα προστεθούν στον χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.

Όποτε πραγματοποιείτε εγγραφή σε ArcGIS Data Store (datastore για σχεσιακά ή χωροχρονικά big data) χρησιμοποιώντας το πτυσσόμενο πλαίσιο Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε μέσα στον φάκελο Το περιεχόμενό μου το όνομα ενός άλλου φακέλου όπου θα αποθηκευτεί το αποτέλεσμα.
