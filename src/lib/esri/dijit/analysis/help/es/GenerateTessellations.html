Generar teselaciones

Esta herramienta genera teselaciones en un área de estudio siguiendo una forma y un tamaño específicos.

La casilla de verificación Usar la extensión del mapa actual está deshabilitada para esta herramienta, ya que la extensión utilizada para generar las teselaciones está controlada por el parámetro Definir extensión.

Elegir forma

La forma o geometría de la teselación que le gustaría crear. Las formas disponibles son: Cuadrado, Hexágono, Hexágono transversal, Triángulo o Diamante.

Definir el área de estudio

La extensión o el área que cubrirá las teselaciones. La extensión se puede determinar con la extensión de mapa actual, la extensión de una capa existente o al dibujar interactivamente un área que se utilizará como extensión.

Si ha seleccionado una capa de entidades para determinar la extensión, se habilitará la opción de conservar solo las teselaciones intersecantes. Si está activada, solamente se generarán en la capa de teselaciones resultante aquellas teselaciones que cubran o intersequen entidades de la capa de extensión especificada.

Nombre de capa de resultados

Este es el nombre de la capa que se creará en Mi contenido y se agregará al mapa. Si la capa ya existe, se le pedirá que indique otro nombre.

Con el menú desplegable Guardar el resultado en, puede especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado.
