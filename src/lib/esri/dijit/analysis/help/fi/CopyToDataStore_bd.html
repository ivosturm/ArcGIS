Kopioi aineistosäilöön

Tämä työkalu kopioi lähtöaineiston karttatason aineistosäilöön. Voit kopioida tulokset ArcGIS Data Storeen (relaatiotietokantaan perustuva tai spatiotemporaalinen Big Data -säilö) ja portaaliisi luodaan karttataso tai voit kopioida tulokset Big Data -tiedostojakoon.

Voit esimerkiksi kopioida Big Data -tiedostoresurssiin tallennettuja kohteita relaatioaineistosäilöön ja määrittää, että vain kohteet nykyisessä kartan laajuudessa kopioidaan. Näin voit luoda isännöidyn kohdepalvelun, joka sisältää vain määritetyssä kartan laajuudessa olleet kohteet.

Valitse kopioitava karttataso

Kopioitava karttataso tai taulukko.

Sen lisäksi, että voit valita karttatason kartasta, voit valita Valitse analyysikarttataso -vaihtoehdon avattavan luettelon alareunasta ja selata Big Data -tiedostoresurssin aineistoryhmän tai karttatason sisältöön.

Käytä kaikkia kohteita tai valitse kohteet (valinnainen)

Voit vaihtoehtoisesti valita kopioitavat sijainnit valintapainikkeella.

Valitse aineistosäilö, johon kopioidaan

Aineistosäilö, johon karttataso kopioidaan. Tämä parametri ohittaa valinnan, joka on tehty kohdassa Analyysiympäristöt. Vaihtoehdot ovat seuraavat:

  • ArcGIS Data Store ArcGIS Data Store
    • Tila-aika: Tämä on oletusarvo.
    • Relaatiotietokantaan perustuva
  • Big Data -tiedostojaon tulosaineistomallit
    • Käytettävissä olevien Big Data -tiedostojakojen luettelo muodossa bigDataFileShareName-templateName. Big Data -tiedostojakojen mallien on oltava määritettynä ennen työkalun suoritusta.

Tulostason nimi

Luotavan karttatason nimi. Jos kirjoitat kohteeseen ArcGIS Data Store, tulokset tallennetaan Oma sisältö -osioon ja lisätään karttaan. Jos kirjoitat Big Data -tiedostojakoon, tulokset tallennetaan Big Data -tiedostojakoon ja lisätään sen manifest-tiedostoon. Tuloksia ei lisätä karttaan. Oletusnimi perustuu työkalun nimeen ja lähtökarttatason nimeen. Jos karttataso on jo olemassa, työkalua ei voi suorittaa.

Kun kirjoitat kohteeseen ArcGIS Data Store (relaatiotietokantaan perustuvaan tai spatiotemporaaliseen Big Data -säilöön) käyttämällä avattavaa Tallenna tulos kohteeseen -luetteloa, voit määrittää kansion nimen Oma sisältö -osiossa, johon tulos tallennetaan.
