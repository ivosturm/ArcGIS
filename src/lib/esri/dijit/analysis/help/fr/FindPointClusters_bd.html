Rechercher des agrégats de points

L’outil Rechercher des agrégats de points détecte les agrégats d’entités ponctuelles dans le bruit environnant en se basant sur leur distribution spatiale.

Par exemple, une organisation non gouvernementale étudie une maladie parasitaire particulière. Vous disposez d’un jeu de données ponctuelles représentant les ménages dans votre zone d’étude, certains d’entre eux étant touchés, et d’autres non. En utilisant l’outil Rechercher des agrégats de points, un analyste peut déterminer les agrégats de ménages infestés pour vous aider à localiser la zone où commencer le traitement et exterminer les parasites.

Choisir la couche pour laquelle les agrégats seront identifiés

La couche de points contenant les agrégats. Les couches doivent se trouver dans une référence spatiale projetée ou disposer d’une référence spatiale projetée définie sur un système de coordonnées projetées à l’aide de l’option Analysis Environments (Environnements d’analyse).

Outre le fait de choisir une couche de votre carte, vous pouvez sélectionner Choose Analysis Layer (Choisir une couche d’analyse) au bas de la liste déroulante pour parcourir votre contenu et rechercher une couche d’entités ou un jeu de données de partage de fichiers Big Data.

Choisissez la méthode d’agrégation à utiliser

Indique la méthode d’agrégation utilisée pour distinguer les agrégats d’entités ponctuelles du bruit environnant. Vous pouvez choisir d’utiliser une distance définie ou un algorithme d’agrégation qui s’ajuste automatiquement.

La méthode de distance définie (DBSCAN) utilise la plage de recherche spécifiée pour séparer les agrégats denses du bruit plus clairsemé. La méthode de distance définie (DBSCAN) est plus rapide, mais elle n’est appropriée que s’il existe une distance de recherche évidente qui s’applique bien à tous les agrégats potentiels. La distance définie (DBSCAN) recherche les agrégats possédant des densités analogues.

Il n’est pas nécessaire d’indiquer une plage de recherche pour la méthode d’ajustement automatique (HDBSCAN), mais cette méthode est plus chronophage. La méthode d’ajustement automatique (HDBSCAN) recherche des agrégats de points similaires à la méthode de distance définie (DBSCAN), mais utilise des plages de recherche variées autorisant des agrégats de densités variables.

Nombre minimal de points à considérer comme agrégat

Ce paramètre est utilisé différemment selon la méthode d’agrégation choisie :

  • Distance définie (DBSCAN) : spécifie le nombre d’entités devant figurer à une certaine distance d’un point pour que ce point commence de former un agrégat. La distance est définie à l’aide du paramètre Limit the search range to (Limiter la plage de recherche à).
  • Ajustement automatique (HDBSCAN) : spécifie le nombre d’entités au voisinage de chaque point (point y compris) à considérer lors de l’estimation de la densité. Ce nombre correspond également à la taille minimale autorisée pour un agrégat lors de l’extraction des agrégats.

Limiter la plage de recherche à

Lors de l’utilisation de la méthode basée sur la distance définie (DBSCAN), ce paramètre désigne la distance au sein de laquelle le paramètre Minimum number of points to be considered a cluster (Nombre minimal de points à considérer comme agrégat) doit figurer. Ce paramètre n’est pas utilisé lorsque l’ajustement automatique (HDBSCAN) est choisi comme méthode d’agrégation.

Nom de la couche de résultat

Nom de la couche à créer. Si vous écrivez sur une instance ArcGIS Data Store, les résultats seront enregistrés dans My Content (Mon contenu) et ajoutés à la carte. Si vous écrivez sur un partage de fichiers Big Data, les résultats seront enregistrés dans le partage de fichiers Big Data et ajoutés à son manifeste. Il ne sera pas ajouté à la carte. Le nom par défaut repose sur le nom de l'outil et sur le nom de la couche en entrée. Si la couche existe déjà, l'outil échoue.

Lors de l’écriture sur ArcGIS Data Store (répertoire de données relationnelles ou spatio-temporelles) via la zone de liste déroulante Save result in (Enregistrer le résultat dans), vous pouvez spécifier le nom d’un dossier dans My Content (Mon contenu) où le résultat sera enregistré.
