העתק למאגר נתונים

כלי זה ייקח שכבת קלט ויעתיק אותה למאגר נתונים. באפשרותך להעתיק תוצאות אל ArcGIS Data Store (מאגר Big Data יחסי או של מרחב-זמן) ותיווצר שכבה בפורטל שלך או להעתיק תוצאות למיקום שיתוף קובצי Big Data.

לדוגמה, תוכל להעתיק ישויות שמאוחסנות במיקום שיתוף קובצי Big Data אל מאגר נתוני יחסי ולציין שרק ישויות עם תיחום מפה נוכחי יועתקו. פעולה זו תיצור שירות ישויות מתארח שכולל רק את הישויות שהיו בתוך תיחום המפה שצוין.

בחר שכבה להעתקה

השכבה או הטבלה שיועתקו.

בנוסף לבחירת שכבה מהמפה שלך, באפשרותך לבחור באפשרות בחר שכבת ניתוח בתחתית הרשימה הנפתחת כדי לדפדף לתכנים שלך עבור סט נתוני שיתוף קובץ Big Data או שכבת ישויות. באפשרותך אופציונלית להחיל מסנן על שכבת הקלט או להחיל בחירה על שכבות מתארחות שנוספו למפה שלך. מסננים ובחירות מוחלים רק עבור ניתוח.

תוכל להשתמש בכל הישויות או לבחור מביניהן (אופציונלי)

אופציונלית, השתמש בלחצן הבחירה כדי לזהות את המיקומים שיועתקו.

בחר את מאגר הנתונים להעתיק אליו

מאגר הנתונים שהשכבה תועתק אליו. פרמטר זה יגבר על הבחירה בסביבות ניתוח. להלן האפשרויות:

  • ArcGIS Data Store ArcGIS Data Store
    • מרחבי-זמני - זוהי ברירת המחדל.
    • יחסי
  • תבניות פלט שיתוף קבצים של Big Data
    • רשימה של תבניות שיתוף קובצי Big Data בפורמט bigDataFileShareName-templateName. יש להגדיר את התצורה של תבניות שיתוף קובצי Big Data לפני הרצת הכלי.

שם שכבת התוצאה

השם של השכבה שתיווצר. אם אתה כותב ל- ArcGIS Data Store, התוצאות שלך יישמרו ב-התוכן שלי ויתווספו למפה. אם אתה כותב לקובץ שיתוף Big Data, התוצאות שלך יאוחסנו בקובץ השיתוף של Big Data ויתווספו למניפסט שלו. הם לא יתווספו למפה. שם ברירת המחדל מבוסס על שם הכלי ושם שכבת הקלט. אם השכבה כבר קיימת, הכלי ייכשל.

בעת כתיב ל- ArcGIS Data Store (מאגר נתונים יחסים או מרחבי-זמני של Big Data) בעזרת התיבה הנפתחת שמור תוצאה ב, באפשרותך לציין את שם התיקייה ב- התוכן שלי שבה התוצאה תישמר.
