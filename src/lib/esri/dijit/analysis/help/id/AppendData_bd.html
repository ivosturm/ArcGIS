Tambahkan Data

Alat ini menambahkan data ke layer fitur yang ada. Tambahkan Data memodifikasi layer input asli dan tidak menghasilkan layer output baru. Anda dapat mencocokkan kolom berdasarkan nama kolom dan jenis kolom, atau Anda dapat menerapkan metode pencocokan yang lebih canggih.

Berikut ini adalah contohnya:

  • Anda adalah karyawan dari perusahaan asuransi besar yang menghasilkan catatan penjualan setiap bulan. Untuk laporan tahunan Anda, data bulanan harus dikompilasi untuk analisis. Untuk memulai analisis penjualan, Anda dapat memanfaatkan Append Data untuk memperbarui layer AnnualReport_2018 Anda setelah nomor bulanan Anda tersedia. Karena Anda tahu organisasi Anda mempertahankan skema yang konsisten setiap bulan, Anda dapat menggunakan metode pemetaan lapangan default untuk secara efisien menambahkan fitur berdasarkan nama dan jenis kolom.
  • Sepuluh kelompok lingkungan masing-masing mengumpulkan ratusan ribu pengukuran iklim dalam yurisdiksi mereka. Organisasi-organisasi ini ingin mengumpulkan data mereka untuk lebih memahami pola regional dan global. Setiap organisasi menentukan skema untuk set data mereka. Mereka masing-masing ditentukan nama kolom yang berbeda untuk suhu dan data kecepatan angin. Mengingat perbedaan nama field antara set data, pemetaan kolom dapat dimanfaatkan untuk menentukan bagaimana cara menambahkan pengukuran suhu dan kecepatan angin ke kolomnya masing-masing.

Pilih layer untuk ditambahkan

Fitur tabel, titik, garis, atau poligon yang datanya akan ditambahkan.

Caution: 

Layer yang Anda tambahkan akan dimodifikasi selama pelaksanaan alat. Sebelum melanjutkan, pastikan Anda ingin memodifikasi layer ini.

Selain memilih layer dari peta, Anda dapat memilih Pilih Layer Analisis di bawah daftar drop-down untuk menelusuri konten Anda untuk feature layer yang di-host.

Pilih layer untuk ditambahkan

Fitur tabel, titik, garis, atau poligon untuk ditambahkan ke layer input.

Selain memilih layer dari peta, Anda dapat memilih Pilih Layer Analisis di bawah daftar drop-down untuk menelusuri konten Anda untuk set data atau feature layer berbagi file big data.

Tambahkan semua fitur atau tentukan subset (opsional)

Mengidentifikasi fitur yang akan ditambahkan. Parameter ini opsional.

Pilih cara menambahkan kolom (opsional)

Menentukan bagaimana kolom layer tambahan ditambahkan ke layer input. Secara default, semua input yang cocok dan kolom tambahan akan ditambahkan. Kolom adalah kecocokan jika mereka memiliki nama kolom yang sama dan jenis kolom yang sama. Secara opsional, Anda dapat memilih bagaimana kolom dicocokkan, atau menghitung ekspresi Arcade untuk mengisi nilai kolom.

Misalnya, Anda memiliki layer input dengan kolom field_number, field_string, date_field, unused_field, dan layer tambahan dengan kolom field_string, field_number, dan MY_DATE_FIELD.

  • Secara default, alat ini akan cocok dengan field_number pada layer input ke field_number pada layer tambahan, dan field_string pada layer input ke field_string pada layer tambahan.
  • Secara default, layer input date_field dan unused_field akan diisi dengan nilai null karena mereka tidak memiliki kolom yang cocok pada layer tambahan.
  • Secara opsional, Anda bisa menambahkan nilai ke kolom yang tidak secara otomatis dicocokkan secara default. Sebagai contoh, date_field dalam layer input dapat dicocokkan dengan kolom tambahan MY_DATE_FIELD. Untuk melakukan ini, gunakan menu drop-down di sebelah date_field di kolom pertama dan pilih MY_DATE_FIELD dari menu drop-down di kolom kedua.
Sebagai contoh lain, layer input memiliki kolom numerik value_1, value_2, value_3, dan my_Average, dan layer tambahan memiliki kolom angka value_1, value_2, dan value_3.
  • Secara default, value_1, value_2, dan value_3 akan dicocokkan dan ditambahkan.
  • Selain perilaku default, nilai dapat dihitung untuk kolom my_Average. Lakukan ini menggunakan beberapa kolom dan operasi matematika dengan memilih Ekspresi pilihan drop-down di kolom di samping my_Average dan membuat ekspresi Arcade kustom seperti 
    average($feature["value_1"] + $feature["value_2"] + $feature["value_3"])
    .
