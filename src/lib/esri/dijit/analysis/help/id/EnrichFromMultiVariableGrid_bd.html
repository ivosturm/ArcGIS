Perkaya dari Multi-Variable Grid

Alat Perkaya dari Multi-Variable Grid secara efisien menggabungkan atribut dari multi-variable grid ke layer titik.

Sebuah multi-variable grid adalah layer apa-pun yang dibuat dengan alat Buat Multi-Variable Grid, yang mengumpulkan atribut dari beberapa layer ke dalam grid poligon tunggal. Menggunakan multi-variable grid untuk pengayaan memungkinkan Anda untuk dengan cepat menambahkan koleksi informasi yang besar dan beragam ke data titik untuk digunakan dalam analisis spasial lebih lanjut.

Contoh, anggaplah Anda telah memperoleh layer peristiwa kejahatan baru yang terjadi sepanjang tahun lalu di seluruh kota. Anda ingin membuat model hubungan antara frekuensi kejahatan dan faktor keragaman demografi, ekonomi, dan geografis. Anda memiliki akses ke data yang menjelaskan faktor-faktor ini, tapi akses ini disimpan dalam lima layer berbeda dengan membedakan jenis geometri. Selain itu, beberapa layer disimpan dalam sebuah berbagi file big data sementara lainnya disimpan sebagai feature service. Dengan membuat multi-variable grid untuk menyimpan informasi ini dan kemudian menggabungkan grid ke peristiwa kejahatan dengan alat Perkaya dari Multi-Variable Grid, Anda dapat secara efisien menggabungkan beragam atribut ke layer titik Anda hanya dengan menjalankan dua alat. Anda kemudian dapat menggunakan atribut yang digabungkan sebagai variabel penjelas dan mulai membuat model frekuensi kejahatan dengan layer hasil.

Pilih fitur titik untuk diperkaya

Fitur titik yang akan diperkaya oleh multi-variable grid.

Selain memilih layer dari peta, Anda dapat memilih Pilih Layer Analisis di bawah daftar drop-down untuk menelusuri konten Anda untuk set data atau feature layer berbagi file big data.

Pilih multi-variable grid untuk diperkaya

Layer multi-variable grid berisi atribut yang akan digabungkan ke fitur titik input. Layer harus dibuat menggunakan alat Buat Multi-Variable Grid dan harus merupakan feature layer yang di-host. Layer multi-variable grid disimpan dalam sebuah berbagi file big data tidak didukung sebagai input pada alat ini.

Pilih variabel yang akan ditambahkan

Atribut dalam layer multi-variable grid yang akan digabungkan ke dalam fitur titik input kapan saja fitur titik memotong grid.

Nama layer hasil

Nama layer yang akan dibuat. Jika Anda menulis ke sebuah ArcGIS Data Store, hasil Anda akan disimpan di Konten Saya dan ditambahkan ke dalam peta. Jika Anda menulis ke berbagi file big data, hasil Anda akan disimpan ke berbagi file big data dan ditambahkan ke manifestasinya. Ini tidak akan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer sudah ada, alat tidak akan berfungsi.
