Genera tassellazioni

Questo strumento genera tassellazioni in un'area di studio sulla base di una forma e di una dimensione specificate.

La casella di controllo Usa estensione mappa corrente è disabilitata per questo strumento perché l'estensione usata per generare tassellazioni è controllata dal parametro Definisci estensione.

Scegli una forma

La forma o geometria della tassellazione che si desidera creare. Le forme disponibili sono Quadrato, Esagono, Esagono trasversale, Triangolo o Rombo.

Definire area di studio

L'estensione o area che sarà coperta dalle tassellazioni. L'estensione può essere stabilita usando l'estensione attuale della mappa, l'estensione di un layer esistente o disegnando in modo interattivo un'area da usare come estensione.

Se si è selezionato un feature layer per determinare l'estensione, sarà abilitata l'opzione per mantenere solo le tassellazioni di intersezione. Se selezionata, nel layer risultante delle tassellazioni saranno generate solo le tassellazioni che coprono o intersecano le feature nel layer di estensione specificato.

Definisci le dimensioni del contenitore specificando

La dimensione di ciascuna cella. La dimensione della cella può essere stabilita specificando unità quadrate o specificando una distanza calcolata adoperando le seguenti misurazioni, in cui d = Distanza:

Square

Esagono

Esagono trasversale

Triangolo

Rombo

Nome del layer dei risultati

Questo è il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Se il layer esiste già, verrà chiesto di fornire un altro nome.

Utilizzare il menù a discesa Salva risultato in per specificare il nome di una cartella in I miei contenuti in cui salvare il risultato.
