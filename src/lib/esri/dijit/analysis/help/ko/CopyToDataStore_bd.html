데이터 저장소로 복사(Copy to Data Store)

이 도구는 입력 레이어를 받아 데이터 저장소에 복사합니다. ArcGIS Data Store(관계형 또는 시공간 빅데이터 저장소)에 결과를 복사할 수 있으며 포털에서 레이어를 만들거나 결과를 빅데이터 파일에 복사할 수 있습니다.

예를 들어 빅 데이터 파일 공유에 저장되어 있는 피처를 관계형 데이터 저장소에 복사하되 현재 맵 범위 내의 피처만 복사되도록 지정할 수 있습니다. 이 경우 지정된 맵 범위 내에 있는 피처만 포함된 호스팅 피처 서비스가 생성됩니다.

복사할 레이어 선택

복사될 레이어나 테이블입니다.

맵에서 레이어를 선택하는 것 이외에, 드롭다운 목록의 하단에 있는 Analysis Layer 선택을 선택하여 빅데이터 파일 공유 데이터셋 또는 피처 레이어의 콘텐츠를 찾아볼 수 있습니다.

모든 피처를 사용하거나 원하는 피처를 선택할 수 있습니다(선택).

필요에 따라 선택 버튼을 사용하여 복사될 위치를 식별할 수 있습니다.

복사할 데이터 저장소 선택

레이어가 복사될 데이터 저장소입니다. 이 매개변수는 분석 환경의 선택 항목을 재정의합니다. 옵션은 다음과 같습니다.

  • ArcGIS Data Store ArcGIS Data Store
    • 시공간형 - 이 옵션이 기본 설정입니다.
    • 관계형
  • 빅데이터 파일 공유 결과 템플릿
    • 사용 가능한 빅데이터 파일 공유 템플릿 bigDataFileShareName-templateName 형식입니다. 빅데이터 파일 공유 템플릿은 도구를 실행하기 전에 구성해야 합니다.

결과 레이어 이름

생성될 레이어의 이름입니다. ArcGIS Data Store에 작성하는 경우 결과는 내 콘텐츠에 저장되고 맵에 추가됩니다. 빅데이터 파일 공유에 작성하는 경우 결과는 빅데이터 파일 공유에 저장되고 해당 매니페스트에 추가됩니다. 이러한 결과는 맵에 추가되지 않습니다. 기본 이름은 도구 이름과 입력 레이어 이름을 기반으로 합니다. 레이어가 이미 있으면 도구를 사용할 수 없습니다.

ArcGIS Data Store(관계형 또는 시공간 빅데이터 저장소)에 작성하는 경우 결과 저장: 드롭다운 상자를 사용하여 내 콘텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
