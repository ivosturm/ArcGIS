Gerar Mosaicos

Esta ferramenta gera mosaicos em uma área de estudo com base na forma e no tamanho especificados.

A caixa de seleção Utilizar extensão de mapa atual é desativada para esta ferramenta, pois a extensão utilizada para gerar os mosaicos é controlada pelo parâmetro Definir extensão .

Escolher forma

A forma ou geometria do mosaico que você gostaria de criar. As formas disponíveis são Quadrado, Hexágono, Hexágono Transversal, Triângulo ou Diamante.

Definir Ã¡rea de estudo

A extensão ou área que será coberta pelo mosaico. A extensão pode ser determinada utilizando a extensão do mapa atual, a extensão de uma camada existente ou o desenho interativo de uma área a ser utilizada como extensão.

Se uma camada de feição tiver sido selecionada para determinar a extensão, a opção de manter unicamente os mosaicos de intersecção será ativada. Se marcada, somente estes mosaicos que cobrem ou interseccionam com feições na camada de extensão especificada serão geradas na camada de mosaico resultante.

Nome da camada resultante

Este é o nome da camada que será criada em Meu Conteúdo e adicionada no mapa. Se a camada já existir, você será solicitado para fornecer outro nome.

Ao utilizar o menu suspenso Salvar resultado em, é possível especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
