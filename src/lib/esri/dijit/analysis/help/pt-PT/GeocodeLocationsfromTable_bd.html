Geocodificar Locais a partir de tabela

Geocodificar converte endereços em coordenadas Esta ferramenta converte endereços a partir de uma tabela de partilha de ficheiros big data numa camada de elementos com as coordenadas como colunas extra.

Selecionar uma tabela de entrada

A tabela de entrada com os endereços a geocodificar.

Selecionar um localizador

Selecione um serviço de localizador em relação ao qual geocodificar. Caso o serviço de localizador que pretende utilizar não se encontre no menu pendente, contacte o seu administrador para que o serviço de localizador seja configurado para geocodificação em batch.

País

Caso todos os endereços sejam do mesmo país, selecione esse país para garantir maior precisão nos resultados. Caso contrário, deixe este parâmetro por preencher.

Selecionar Campos do Dados

O serviço de localizador exige a inclusão de certos campos nos dados, tais como endereço, cidade, estado e código postal ou código ZIP. Faça corresponder os campos de dados no seu ficheiro ou tabela de entrada aos campos que o localizador exige, recorrendo às caixas pendentes. Quantos mais campos cartografar, maior será a precisão dos resultados. Caso todos os dados da tabela de entrada se encontrem numa só coluna, selecione Campo Único, e selecione um campo de dados a partir do menu pendente. Caso contrário, selecione Múltiplos Campos, e cartografe os campos de localizador apropriados às colunas de múltiplos campos da sua tabela.

Para cada localização

Inclua campos adicionais devolvidos pelo serviço de localização. Caso esta opção esteja selecionada, todos os campos criados pelo localizador serão devolvidos. Caso não esteja selecionada, os seus dados de entrada serão devolvidos com as coordenadas geocodificadas.

Nome do resultado de saída

O nome da camada que será criada. Se estiver a escrever para ArcGIS Data Store, os seus resultados serão gravados em O Meu Conteúdo e adicionados ao mapa. Se estiver a escrever para uma partilha de ficheiros big data, os seus resultados serão guardados numa partilha de ficheiros big data e adicionados ao seu manifesto. Não serão adicionados ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Caso a camada já existe, a ferramenta irá falhar.

Ao escrever para ArcGIS Data Store (armazenamento relacional ou espaciotemporal de dados big data) utilizando a caixa suspensa Guardar resultado em pode especificar o nome de uma pasta em O Meu Conteúdo, onde será armazenado o resultado.
