Summarize Data

The Summarize Data toolset contains tools that calculate total counts, lengths, areas, and basic descriptive statistics of features and their attributes within areas or near other features.

  • Aggregate Points calculates statistics about points that fall within specified areas or bins.
  • Build Multi-Variable Grid generates a grid of square or hexagonal bins and calculates values for each bin based on the proximity of each input layer.
  • Describe Dataset calculates a summary of your input, and can optionally generate a sample of your input layer.
  • Join Features calculates statistics about features that share a spatial, temporal, or attribute relationship with other features.
  • Reconstruct Tracks calculates statistics about points or polygons that belong to the same track and reconstructs inputs into tracks.
  • Summarize Attributes calculates statistics about feature or tabular data that share attributes.
  • Summarize Within calculates statistics for area features and attributes that overlap each other.

Агрегировать точки

Используя слой точечных объектов и либо слой площадных объектов, либо бины, определяемые заданным расстоянием, инструмент определяет, какие точки укладываются в каждую область или бин, и вычисляет статистику по всем точкам каждой области или бина. Дополнительно можно применить к этому инструменту временные срезы.

Ниже приведены примеры:

  • Используя точки мест правонарушений, можно вычислить число правонарушений в каждом округе или другой административной единице.
  • Найдите максимальную и минимальную месячную выручку среди франчайзи, используя 100-m-километровые бины.


Построить сеть по нескольким переменным

Инструмент Построить сеть по нескольким переменным создает сетку квадратных или шестиугольных ячеек и вычисляет значения в каждой из них на основании близости входных слоев.

Ниже приведены примеры:

  • Как на основе данных об инфраструктуре общественного транспорта понять, какая часть города является наименее доступной?
  • Как на основе данных о водных объектах (реках и озерах) узнать, какой из них является ближайшим к каждой точке на территории США?
  • Как на основе данных о доходах понять, где в США наибольшие различия по этому показателю в пределах 50 миль?


Описание набора данных

Выводит статистику о входных объектах, создает слои с выбранными объектами и визуализацию данных. Вы можете выбрать в качестве выходных данных слой выбранных объектов или слой со всем экстентом данных.

Ниже приведены примеры:

  • Чтобы понять, какие значения содержит набор данных, обратитесь к вычисленной сводной статистике.
  • Вместо того, чтобы отображать весь набор данных, создайте поднабор слоев для визуализации объектов и их атрибутов, когда они добавлены на карту.
  • Экономьте время и ресурсы и выполняйте анализ на слое выбранных объектов перед выполнением анализа на всем наборе данных.


Присоединить объекты

Используя слои объектов или табличные данные можно соединить объекты и записи на основе определенных отношения между входными слоями или таблицами. Присоединения будут определяться пространственными, временными и атрибутивными отношениями, а также дополнительно вычисляемой суммарной статистикой.

Ниже приведены примеры:

  • Имея точечные местоположения случаев преступлений за период времени, соедините данные о преступлениях с ними же, указав пространственные отношения преступлений в пределах 1 километра друг от друга и случившихся в пределах 1 часа друг от друга, чтобы определить, происходили ли последовательности преступлений в сравнительной близости по расстоянию и времени.
  • Имея таблицу почтовых кодов с демографической информацией и площадные объекты, представляющие жилые дома, соедините демографическую информацию к информации о проживании, так что каждое здание будет содержать информацию.


Заново создать треки

Используя слой точечных либо полигональных объектов с информацией о времени, которые представляют текущий момент времени, этот инструмент определяет, какие входные объекты принадлежат треку и последовательно упорядочивает входные объекты по времени. Дополнительно вычисляется статистика входных объектов внутри каждого трека.

Ниже приведены примеры:

  • Имея точечные местоположения и времена измерения урагана, вычислите среднюю скорость ветра, а также максимальное давление ветра урагана.
  • Используйте GPS-данные с кораблей чтобы восстановить их маршруты в виде линий.


Суммировать атрибуты

Используя данные объектов, либо табличные данные, этот инструмент суммирует статистику указанных полей.

Ниже приведены примеры:

  • Имея местоположения продуктовых магазинов с полем COMPANY_NAME, суммируйте магазины по имени компании, которой они принадлежат, чтобы определить статистику для каждой компании.
  • Имея таблицу продуктовых магазинов с полями COMPANY_NAME и COUNTY, суммируйте магазины по имени компании и округу, чтобы определить статистику для каждой компании в каждом округе.


Суммировать в пределах

Находит области (или их части), которые пересекаются между двумя слоями и вычисляет статистику пересечения.

Ниже приведены примеры:

  • Используя слой водоразделов и слой с типами землепользования, можно определить общую площадь земель различных типов для каждого водораздела.
  • Используя слой земельных участков в округе и слой с границами городов, можно вычислить среднюю стоимость свободных участков в каждом городе.


Найти местоположения

Эти инструменты используются для определения областей, соответствующих нескольким заданным критериям.

  • Выявление инцидентов работает с временными слоями для определения объектов, соответствующим заданным критериям.
  • Геокодировать адреса из таблицы конвертирует адреса в координаты
  • Найти схожие местоположения находит по заданным критериям места, наиболее схожие с одним или несколькими базовыми местоположениями.

Выявление инцидентов

Этот инструмент работает со слоем с поддержкой времени, содержащим точечные, линейные, полигональный объекты или таблицы, которые представляют текущий момент времени. Используя последовательно упорядоченные объекты, называемые треками, этот инструмент определяет объекты, являющиеся интересующими вас инцидентами. Инциденты определяются по заданным вами условиям.

Ниже приведены примеры:

  • Определение показателей урагана, относящегося к 4 категории, на основе скорости ветра.
  • Используя GPS-данные движения кораблей с полем скорости, найдите измерения, в 1.5 раза превышающие среднее значение первых пяти измерений.


Геокодирование адресов из таблицы

Конвертирует адреса в координаты. Используйте этот инструмент с таблицами файлового хранилища больших данных.


Найти схожие местоположения

На основе выбранного вами критерия, определите схожие местоположения, измерив схожесть местоположений в слое поиска кандидатов с одним или несколькими опорными местоположениями.

Ниже приведены примеры:

  • Найдите десять наиболее похожих магазинов, изучив количество сотрудников и объемы продаж за год.
  • Найдите 100 наиболее похожих городов, изучив отношения между населением, ежегодным приростом и подоходным налогом.


Анализ структурных закономерностей

Эти инструменты помогут вам идентифицировать, оценить значимость и визуализировать пространственные закономерности в распределении своих данных.

  • Вычислить плотность производит отбор определенного числа неких явлений и распределяет их по карте.
  • Инструмент Найти горячие точки определяет наличие статистически значимой кластеризации пространственной закономерности в распределении данных.
  • Поиск кластеров точек определяет среди точечных объектов кластеры в окружающем шуме на основе их пространственного распределения.
  • Классификация на основе леса и регрессия создает модели и строящий прогнозы при помощи адаптации метода контролируемого машинного обучения произвольного леса Лео Бреймана.
  • Обобщенная линейная регрессия строит обобщенную линейную регрессию для вычисления прогнозов или моделирования взаимосвязи между независимыми переменными и зависимой переменной.

Вычислить плотность

Инструмент Подсчитать плотность создает карту плотности из точечных пространственных объектов посредством разброса по карте известного количества событий (представленных как атрибуты точек). В результате вы получаете слой областей, представляющих плотность.

Ниже приведены примеры:

  • Расчет плотности больниц в административном районе. Слой результатов покажет районы с высокой и низкой доступностью больниц, и эта информация может быть использована при принятии решения о том, где необходимо строить новые больницы.
  • Определение районов с высоким уровнем риска возникновения лесных пожаров на основе исторических данных о местах их возникновения.
  • Поиск удаленных от основных дорог населенных пунктов для планирования мест, где должны быть построены новые дороги.


Поиск кластеров точек

Инструмент Поиск кластеров точек определяет среди точечных объектов кластеры в окружающем шуме на основе их пространственного распределения.

Ниже приведены примеры:

  • Поиск кластеров среди инфицированных домохозяйств, чтобы провести целевую вакцинацию в очагах инфекции.
  • Информируйте и предпринимайте эвакуационные и спасательные меры, основываясь на размере и расположении кластеров, используя геопривязанные твиты с сообщениями о природных катастрофах или террористических атаках.


Найти горячие точки

Инструмент Найти горячие точки определяет наличие статистически значимой кластеризации в пространственном распределении данных.

  • Действительно ли точки (места правонарушений, деревья, ДТП) кластеризованы? Как в этом убедиться?
  • Действительно ли вы нашли статистически значимую горячую точку (для затрат, детской смертности, постоянно высоких результатов школьных экзаменов), или при изменении способа присвоения символов ситуация на карте изменяется?
Инструмент Найти горячие точки позволяет точно ответить на эти вопросы.


Классификация на основе леса и регрессия

Инструмент Классификация на основе леса и регрессия создает модели и строящий прогнозы при помощи адаптации метода контролируемого машинного обучения произвольного леса Лео Бреймана

Ниже приведены примеры:

  • Имеются данные данные о наличии морских водорослей, а также ряд экологических независимых переменных, представленных атрибуты, которые были обогащены с помощью сетки с несколькими переменными для вычисления расстояний до заводов вверх по течению и крупных портов. Появление морских водорослей в будущем может быть предсказано на основе прогнозов для тех же экологических независимых переменных.
  • Предположим, у вас есть данные о урожайности на сотнях ферм по всей стране, а также другие атрибуты на каждой из этих ферм (количество сотрудников, площадь в акрах и т.п.) Используя фрагменты этих данных, вы можете предоставить набор объектов, представляющих фермы, урожайность на которых неизвестна (но имеются все остальные переменные), и выполнить прогнозирование урожайности.
  • Стоимость жилья можно спрогнозировать на основе цен домов, которые были проданы в текущем году. Цена домов при продаже, а также информация о числе спален, расстоянии до школ, близости автомагистралей, среднем доходе и числе преступлений могут быть использованы для прогноза продажной стоимости аналогичных домов.


Обобщенная линейная регрессия

Инструмент Обобщенная линейная регрессия строит обобщенную линейную регрессию для вычисления прогнозов или моделирования взаимосвязи между независимыми переменными и зависимой переменной. Инструмент используется для подгонки различных моделей, в частности, непрерывных (МНК), бинарных (логистических) и числовых (Пуассона).

Ниже приведены примеры:

  • Какие демографические характеристики влияют на высокую степень использования общественного транспорта?
  • Имеется ли положительная взаимосвязь между вандализмом и взломами?
  • Какие переменные эффективно предсказывают количество звонков в службу быстрого реагирования? С учетом тенденции, какие ресурсы необходимы для реагирования на экстренные вызовы?
  • Какие факторы обуславливают низкий уровень рождаемости


По Близости

Эти инструменты помогут ответить на самый популярный вопрос пространственного анализа: "Что находится рядом с чем?"

  • Создать буферы создает полигоны вокруг объектов, находящиеся от них на определенном расстоянии.

Создать буферы

Буфер – это область, располагающаяся на указанном расстоянии от точечного, линейного или полигонального объекта.

Ниже приведены примеры:

  • Используя линейные объекты рек, создайте буфер вокруг каждой реки размером, равным 50-кратной ширине реки, чтобы определить предложенную прибрежную границу.
  • Имея области, представляющие страны, создайте буфер вокруг каждой страны размером в 200 морских миль, чтобы определить морскую границу.


Manage Data

These tools are used for the day-to-day management of geographic and tabular data.

  • Append Data appends features to an existing hosted feature layer.
  • Calculate Field creates a new field or modifies an existing field and creates a new layer in your contents.
  • Clip Layer copies features that intersect a defined area of interest.
  • Copy to Data Store copies data to your ArcGIS Data Store and creates a layer in your contents.
  • Dissolve Boundaries merges areas that share a common boundary or attribute value.
  • Merge Layers copies all features from two layers into a single result layer.
  • Overlay Layers combines two or more layers into a single layer.

Копировать в хранилище данных

Копирует входной векторный слой или таблицу в ArcGIS Data Store и создает слой в ArcGIS Enterprise.

Ниже приведены примеры:

  • Скопируйте набор файлов .csv из файлового хранилища больших данных в хранилище пространственно-временных данных для визуализации.
  • Скопируйте объекты из текущего экстента карты, находящиеся в хранилище пространственно-временных данных, в реляционное хранилище данных.


Вычислить поле

Вычисляет значения нового или существующего поля и создает слой в таблице содержания в ArcGIS Enterprise.

Ниже приведены примеры:

  • Измените текущее имя поля total, в котором теперь будет храниться суммарная выручка из полей total_2016, total_2017 и total_2018.
  • Создайте новое поле для категоризации уровней опасности на основе значений в полях windspeed и pollutant.


Вырезать слой

Вырезает входные объекты из указанных областей интереса. В результате получается поднабор входных объектов.

Ниже приведены примеры:

  • Создание поднабора зданий, которые находятся в зонах затопления и могут быть затоплены.
  • Определение, какие большие дороги проходят через потенциальные области застройки.


Слияние границ

Выполняет слияние площадных объектов, которые имеют пересечения границ или делят общее значение поля, чтобы создать непрерывные или многосоставные объекты.

Ниже приведены примеры:

  • Создание единого объекта, который представляет собой штат Гавайи путем слияния границ Гавайских островов.
  • Объединить схожие области изучения на основе общего типа почвы, чтобы в результате получить набор данных непрерывных объектов с одним типом почвы.


Наложение слоев

Соединяет два или более слоев в один. Наложение используется для ответа на один из самых главных вопросов географии: Что находится поверх чего?

Ниже приведены примеры:

  • Какие участки находятся в пределах зоны, подвергавшейся затоплению в последние 100 лет? ("В пределах" – это другой способ сказать "поверх".)
  • Какой тип землепользования соответствует какому типу почвы?
  • Какие скважины находятся в пределах заброшенных военных баз?


Присоединить данные

Присоединяет объекты к существующему размещенному слою в ваших ресурсах в ArcGIS Enterprise.

Ниже приведены примеры:

  • Обновляйте имеющийся векторный слой 2018_Sales каждый месяц по мере выхода отчетов.
  • Присоедините наборы данных по отслеживанию ураганов за 2016 и 2017 годы в набор мастер-данных по ураганам.
  • Собирайте и присоединяйте наборы данных из разных источников с разными схемами с помощью пользовательского сопоставления полей.


Слияние слоев

Комбинирует два векторных слоя для создания одного результирующего слоя. По умолчанию все поля из присоединяемого слоя также будут добавлены, но вы можете настроить пользовательские правила для присоединяемых полей.

Ниже приведены примеры:

  • Слияние тех слоёв, содержащих демографическую информацию из Англии, Уэльса и Шотландии, для создания единого слоя для Великобритании.
  • Слияние двух слоев, содержащих информацию об земельных участках для смежных поселков, при сохранении только полей, имена и типы данных которых совпадают в обоих слоях.


Обогащение данных

Эти инструменты помогают выявить характеристики различных областей.

  • Обогатить из сетки нескольких переменных присоединяет атрибуты из слоя сетки с несколькими переменными к точечному слою.

Обогатить из Сетки нескольких переменных

Эффективно объединяет атрибуты из сетки нескольких переменных в точечный слой, что позволяет быстро добавлять большой и разнообразный набор информации в точечные данные для дальнейшего пространственного анализа.

Ниже приведены примеры:

  • Если у вас есть слой, содержащий миллионы инцидентов отключения электроэнергии, обогатите характеристики инцидента информацией о типичном использовании, экологических рисках и условиях инфраструктуры, чтобы изучить связь между этими факторами и частотой отключения электроэнергии.
