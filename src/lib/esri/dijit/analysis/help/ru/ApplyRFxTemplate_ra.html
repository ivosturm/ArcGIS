Применить шаблон растровой функции

Этот инструмент обрабатывает изображения с помощью функции или последовательности функций, как указано в шаблоне растровой функции.

Входные данные

Позволяет выбирать слои для обработки и настраивать любые параметры, связанные с растровыми функциями в последовательности обработки.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в.
