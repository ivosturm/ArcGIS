Описание набора данных

Инструмент Описание набора данных предоставляет более подробные сведения в наборе данных. Перед запуском анализа большого количества данных, используйте этот инструмент, чтобы определить, что представляют собой входные данные и где они находятся. По умолчанию, инструмент создает строку JSON с ключевыми сведениями о входном слое и таблицу с суммарной статистикой по каждому полю.

Дополнительно вы можете создать выходные векторные слои для более полного описания данных. Это может быть слой образцов или слой экстента. Можно выбрать один из них или оба слоя.

Например, вам предоставлено файловое хранилище больших данных, содержащее 15 наборов данных. Каждый набор данных содержит 10 миллионов полигональных объектов, обозначающих здания и дома по различным регионам страны. Вам необходимо проанализировать регион C, но вы не знаете, в каком наборе данных находятся эти объекты. Чтобы выяснить, какой набор данных вам нужен, запустите инструмент Описание набора данных и выясните, какой набор данных соответствует изучаемой области.

Или представьте, вы разрабатываете процесс анализа большого объема данных. Вы хотите испытать рабочий процесс, но обработка всех данных займет часы или дни. Вместо этого, создайте пробный слой, чтобы эффективно проверить рабочий процесс.

Выберите набор данных для описания

Слой, содержащий точечные, линейные, полигональные или табличные объекты, которые будут обрабатываться.

Кроме выбора слоя из вашей карты, можно щелкнуть Выбрать слой анализа в нижней части ниспадающего списка, чтобы найти свои ресурсы для набора данных или векторного слоя файлового хранилища больших данных.

Изучить набор данных, создав (дополнительно)

Дополнительные выходные слои, помогающие лучше изучить большие данные. Можно выбрать один или оба из этих слоев:

  • Слой экстента - создает полигональный объект, представляющий собой экстент входных объектов области интереса.
  • Образец слоя - создает слой поднабора, содержащий указанное количество входных объектов из набора данных или области интереса.

Образец слоя

Создает слой, содержащий поднабор объектов из входного слоя. Если выбран Образец слоя, можно задать количество объектов, которое необходимо получить в образце слоя. По умолчанию в выходном слое содержится 100 объектов.

Значение должно быть больше нуля. Если задать число, превышающее общее число объектов, будут возвращены все объекты.

Если выбрано Использовать текущий экстент карты, образец слоя будет содержать объекты с текущего экстента карты.

Экстент слоя

Создает векторный слой, содержащий экстент входных объектов, с помощью кнопки Экстент слоя. Эта опция выбрана по умолчанию.

Если выбрано Использовать текущий экстент карты, слоя экстента будет содержать видимый экстент карты.

Имя слоя результата

Имя создаваемого слоя. Если вы записываете в ArcGIS Data Store, ваши результаты будут сохранены в Моих ресурсах и добавлены на карту. Если вы записываете в файловое хранилище больших данных, ваши результаты будут сохранятся в файловом хранилище больших данных и добавляться в его файл манифеста. Они не будут добавлены на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если слой уже существует, произойдет сбой.
