Слияние слоев

Этот инструмент комбинирует два векторных слоя для создания одного результирующего слоя. Слои для слияния должны иметь одинаковый тип геометрии и тип данных времени. Способом объединения и копирования полей можно управлять. Пример:

  • У вас имеется три слоя: Англия, Уэльс и Шотландия, а вы хотите создать один слой - Великобритания.
  • У вас есть 2 слоя, каждый из которых содержит информацию о земельных участках в смежных поселках (участок площадью 36 кв. миль). Я хочу объединить их в один слой, сохранив только поля с одинаковыми названиями и типами из всех 2 слоев.

Выбрать слой

Слой с табличными, точечными, линейными или площадными объектами, для которого будет выполнено слияние со вторым векторным слоем. Все поля из этого слоя будут перенесены в выходной слой.

Выберите слой для слияния

Векторный слой для слияния с входным слоем. Слой для слияния должен иметь тот же тип геометрии и тип данных времени, что и у входного слоя.

Обновить объединенные поля

По умолчанию все поля обоих слоев будут включены в выходные данные. Вы можете изменить способ включения полей из слоя для слияния в выходные данные с помощью следующих параметров:

  • Переименовать – поле в выходных данных будет переименовано.
  • Удалить – поле будет удалено из выходных данных.
  • Сопоставить – имена полей приведены в соответствие друг другу. Значения поля из слоя для объединения будут скопированы в соответствующие выходные поля.

Если оба слоя содержат поле с идентичным именем и типом данных, содержимое этих полей будет объединено в одно поле с тем же именем. Это не применимо к полям, где используется Сопоставление в параметре Обновить объединенные поля.

Имя слоя результата

Имя создаваемого слоя. Если вы записываете в ArcGIS Data Store, ваши результаты будут сохранены в Моих ресурсах и добавлены на карту. Если вы записываете в файловое хранилище больших данных, ваши результаты будут сохранятся в файловом хранилище больших данных и добавляться в его файл манифеста. Они не будут добавлены на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если слой уже существует, произойдет сбой.

При записи в ArcGIS Data Store (реляционное или пространственно-временное хранилище больших данных) с помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
