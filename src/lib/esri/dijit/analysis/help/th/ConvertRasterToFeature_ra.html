แปลงแรสเตอร์เป็นเวกเตอร์

เครื่องมือนี้แปลงแรสเตอร์เป็นฟีเจอร์ชุดข้อมูล, เช่น จุด, เส้นหรือรูปแบบของพื้นที่

ส่วน เลือกฟิลด์ พารามิเตอร์ที่ช่วยให้คุณสามารถเลือกฟิลด์แอทริบิวต์ของชุดข้อมูลที่ป้อนข้อมูลแรสเตอร์แอตทริบิวต์จะกลายเป็นแอตทริบิวต์ในระดับคุณลักษณะการส่งออก พารามิเตอร์นี้จะใช้ได้เฉพาะแรสเตอร์จำนวนเต็มที่มีแอตทริบิวต์เพิ่มเติมในตารางที่เกี่ยวข้อง

แรสเตอร์ที่ใส่เป็นอินพุตสามารถมีขนาดเซลล์เท่าใดก็ได้

แรสเตอร์จำนวนเต็มสามารถแปลงเป็นข้อมูลจุด, เส้น หรือพื้นที่รูปปิด; อย่างไรก็ตาม แรสเตอร์ที่เป็นจุดทศนิยมจะสามารถแปลงเป็นข้อมูลจุดได้เท่านั้น.

ถ้า ใช้ขอบเขตแผนที่ปัจจุบัน ถูกเช็ค, เฉพาะเซลล์แรสเตอร์ที่แสดงผลภายในขอบเขตแผนที่ปัจจุบันจะถูกแปลง ถ้าไม่ได้เช็ค, ค่าเซลล์ทั้งหมดของชั้นข้อมูลแรสเตอร์จะถูกแปลง, ถึงแม้ว่าค่าเซลล์เหล่านั้นอยู่นอกขอบเขตแผนที่ปัจจุบันก็ตาม บันทึกนี้ถ้าชั้นแรสเตอร์ที่มีขนาดใหญ่หรือมีความละเอียดสูงมาก

กรณีการใช้เครื่องมือนี้รวมถึงต่อไปนี้:

  • สร้างข้อมูลพื้นที่รูปปิดจากการจำแนกการใช้ประโยชน์ที่ดินของภาพแรสเตอร์
  • เส้นทางน้ำที่ระบุในแบบจำลองความสูงสามารถแปลงเป็นข้อมูลเส้นได้
  • แบบจำลองความสูงสามารถแปลงเป็นข้อมูลจุดที่สามารถนำมาใช้ในขั้นตอนการทำงานที่ไม่ใช่แรสเตอร์ก็ตาม

เลือกชั้นข้อมูลแรสเตอร์ที่ต้องการแปลง

ใส่ชั้นข้อมูลแรสเตอร์ที่จะแปลงเป็นชั้นข้อมูลฟีเจอร์

เลือกฟิลด์

ใส่ฟิลด์ที่กำหนดค่า value ที่จะใช้สำหรับในการแปลง

มันอาจจะเป็นจำนวนเต็มหรือข้อความใด ๆ

ฟิลด์ที่ประกอบด้วยค่าจุดทศนิยมเท่านั้นที่จะสามารถใช้ในการส่งออกเป็นชุดข้อมูลจุด

ทางเลือกเริ่มต้นคือการใช้ฟิลด์ Value ซึ่งประกอบด้วยค่าที่อยู่ในแต่ละเซลล์แรสเตอร์

เลือกประเภทของผลลัพธ์

เลือกประเภทของผลลัพธ์ระหว่างจุด, เส้น, หรือพื้นที่รูปปิด.

  • จุด —แรสเตอร์จะแปลงออกมาเป็นชุดข้อมูลจุด
  • เส้น —แรสเตอร์จะแปลงออกมาเป็นชุดข้อมูลเส้น
  • พื้นที่รูปปิด —แรสเตอร์จะแปลงออกมาเป็นชุดข้อมูลพื้นที่รูปปิด

ลดความซับซ้อนของเส้นหรือรูปหลายเหลี่ยม

กำหนดถ้าต้องการลดความซับซ้อนของฟีเจอร์

ถ้าเช็ค (ค่าตั้งต้น), ข้อมูลเส้นหรือพื้นที่รูปปิดจะสมูทมากกว่าผลลัพธ์ทั่วไป ถ้าตัวเลือกนี้ไม่ได้เช็ค, ข้อมูลเชิงเส้นจะเป็นไปตามขอบเขตของเซลล์แรสเตอร์ดาต้าเซ็ต

ชื่อของชั้นข้อมูลผลลัพธ์

ชื่อของชั้นข้อมูลที่จะถูกสร้างขึ้นใน เนื้อหาของฉัน และเพิ่มไปยังแผนที่ ชื่อเริ่มต้นจะขึ้นอยู่กับชื่อเครื่องมือและชื่อชั้นข้อมูลอินพุต ถ้าชั้นข้อมูลนี้มีอยู่แล้ว คุณจะถูกขอให้ตั้งชื่อใหม่
