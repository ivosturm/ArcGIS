Пошук подібних місць розташування

На основі вказаних критеріїв інструмент «Пошук подібних місць розташування» вимірює подібність місць розташування у вашому шарі пошуку кандидатів з одним або декількома опорними місцями розташування. Цей інструмент допомагає відповісти на наступні запитання:

  • Які з ваших магазинів найбільш подібні до ваших магазинів з найкращими показниками щодо профілів клієнтів?
  • На основі характеристик сіл, які найбільше постраждали від захворювання, які інші села знаходяться в зоні високого ризику?
Для відповіді на подібні запитання необхідно надати опорні місця розташування, місця розташування для пошуку кандидатів та поля, які представляють критерії, для яких треба знайти відповідність. Перший вхідний шар повинен містити ваші опорні або еталонні місця розташування. Наприклад, це може бути шар, який містить магазини з найкращими показниками або села, які найбільше постраждали від захворювання. Після цього треба вказати шар, який містить ваші місця розташування для пошуку кандидатів. Це можуть бути всі ваші магазини або всі інші села. В кінці необхідно визначити одне або декілька полів, які будуть використовуватись для вимірювання подібності. Після цього інструмент «Пошук подібних місць розташування» виконає оцінку всіх місць розташування для пошуку кандидатів, враховуючи те, наскільки вони відповідають вашим еталонним місцям розташування по всіх вибраних полях.

Вибрати шар, який містить опорні місця розташування

Шар точок, ліній або областей, який містить опорні місця розташування, для пошуку відповідності.

In addition to choosing a layer from your map, you can select Choose Analysis Layer found at the bottom of the drop-down list. This option opens a gallery where you can browse to and select datasets from various sources to use in your analysis.

Можна використовувати всі місця розташування або зробити вибір

За допомогою кнопок вибору визначте опорні місця розташування, за необхідності. Наприклад, якщо вхідний шар містить всі місця розташування — опорні місця розташування, а також місця розташування для пошуку кандидатів — необхідно використовувати один з інструментів вибору для визначення опорних місць розташування. Якщо ви створите два окремих шари, один з опорними місцями розташування і один з усіма місцями розташування для пошуку кандидатів, робити вибір не треба.

Якщо є більш ніж одне опорне місце розташування, створюється одиночне місце розташування шляхом усереднення значень для кожного з полів, які використовуються для аналізу подібності.

Пошук подібних місць розташування в

Місця розташування для пошуку кандидатів в цьому шарі будуть оцінюватись від найменш до найбільш подібних. Подібність базується на тому, наскільки близько місце розташування збігається з опорним місцем розташування по вказаним полям.

In addition to choosing a layer from your map, you can select Choose Analysis Layer found at the bottom of the drop-down list. This option opens a gallery where you can browse to and select datasets from various sources to use in your analysis.

Визначати подібність на основі

Вибрані вами поля будуть виступати в якості критерії для оцінки подібності. Якщо вибрати, наприклад, поле населення та дохід, місцями розташування для пошуку кандидатів з найнижчою (найкращою) оцінками будуть ті місця, які мають значення для населення та доходу, подібні до ваших опорних місць розташування

Показати мені

Ви можете або переглянути всі місця розташування для пошуку кандидатів, ранжировані від найбільш подібних до найменш подібних, або ви можете вказати кількість результатів, яке треба вивести.

  • Всі місця розташування від найбільш до найменш подібних — всі об'єкти в шарі пошуку кандидатів будуть включені в оціночний порядок в результуючому шарі
  • Число повернутих місць розташування — ви можете визначити, скільки найбільш подібних кандидатів повинні бути включені в результуючий шар. Вхідне значення за замовчуванням: 1.

Назва результуючого шару

Надайте назву шару, який буде створено в Моєму змісті та додано до карти. Цей результуючий шар буде містити опорні місця розташування та кількість оцінених місць розташування для пошуку кандидатів, вказаних вами. Якщо назва результуючого шару вже існує, вам буде запропоновано перейменувати його.

За допомогою розкривного списку Зберегти результат в можна вказати назву папки у Моєму змісті, куди буде збережено результат.
