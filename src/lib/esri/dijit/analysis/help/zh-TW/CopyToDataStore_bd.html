複製到資料儲存庫

此工具會採用輸入圖層，並將圖層複製到資料儲存庫。您可以將結果複製到 ArcGIS Data Store（關聯式或大型的時空資料儲存庫），隨即將在您的入口網站中建立圖層，或可以將結果複製到大數據檔案。

例如，您可以將 大數據檔案分享儲存的圖徵複製到 關聯式資料儲存庫，並指定僅複製目前地圖範圍內部的圖徵。這會建立託管圖徵服務，該服務只包含指定的地圖範圍內部的圖徵。

選擇要複製的圖層

將複製的圖層或表格。

除了從地圖選擇圖層外，您也可以選擇下拉式清單底部的 選擇分析圖層，以瀏覽到大數據檔案分享資料集或圖徵圖層的內容。

您可以使用所有圖徵或進行選擇 (選用)

（選用）使用選擇按鈕來識別要複製的位置。

選擇要複製到的資料儲存庫

要將圖層複製到的資料儲存庫。 此參數將覆寫 分析環境中的選擇。 以下是選項：

  • ArcGIS Data Store ArcGIS Data Store
    • 時空 - 這是預設值。
    • 關聯式
  • 大數據檔案分享輸出樣板
    • 格式為 bigDataFileShareName-templateName 的可用大數據檔案分享樣板。必須先配置大數據檔案分享樣板再執行工具。

結果圖層名稱

將建立之圖層的名稱。若是寫入到 ArcGIS Data Store，您的結果將儲存於我的內容並新增至地圖中。若是寫入到大數據檔案分享，您的結果將儲存於大數據檔案分享並新增至其資訊清單中。不會將它新增到地圖。預設名稱參照工具名稱以及輸入圖層名稱。如果圖層已存在，工具將失敗。

使用 將結果儲存到下拉式方塊寫入到 ArcGIS Data Store（關聯式或大型的時空資料儲存庫）時，可以在我的內容中指定將儲存結果的資料夾名稱。
