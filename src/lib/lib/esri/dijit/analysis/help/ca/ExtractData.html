Extreu les dades

Amb aquesta eina, podeu seleccionar i baixar les dades d'una àrea d'interès especificada. Les capes que seleccioneu s'afegiran a un fitxer ZIP, CSV o KMZ.

Capes que s'extrauran

Aquestes són les capes de la taula de continguts de les quals podeu extreure entitats. Les capes activades s'extreuen i s'inclouen al fitxer ZIP o CSV resultant.

Perquè una capa aparegui a la llista de capes que s'extrauran, heu de ser el propietari de les dades o bé el propietari de les dades ha d'habilitar-ne l'exportació. Per habilitar l'exportació de dades de la vostra propietat:

  • Obriu la descripció de l'element de les dades.
  • Feu clic a la pestanya Configuració.
  • Desplaceu-vos cap avall fins a la secció Exporta les dades.
  • Activeu l'opció Permet que altres persones puguin fer exportacions a diferents formats. Quan acabeu, feu clic a Desa.

Si la vostra capa d'anàlisi conté fitxers adjunts, tingueu en compte que només s'extrauran si el format de les dades de sortida és una geobase de dades de fitxers i l'opció Retalla entitats està desactivada.

Àrea que defineix quines entitats s'inclouran al fitxer ZIP o CSV de sortida. Podeu triar entre les opcions següents:

  • L'extensió visible actual del mapa ( Igual que la visualització).
  • Una de les capes de la taula de continguts. Si la capa seleccionada es d'entitats de punts o línies, l'extensió de la capa s'utilitzarà com a àrea d'estudi.
  • Dibuixeu una àrea per definir l'àrea d'estudi.
S'extrauran totes les entitats que estiguin totalment dins de l'extensió del mapa. Per a les entitats que creuen els límits de l'extensió del mapa o s'hi intersequen, però no estan completament dins de l'extensió, teniu dues opcions:
  • Seleccioneu entitats: s'extrauran les entitats completes que creuin el límit de l'àrea d'estudi.
  • Retalla entitats: només s'extrauran les parts de les entitats que estiguin dins de l'àrea d'estudi.

Format de les dades de sortida

Format de les dades que s'extrauran. Quan s'exporten entitats de línies o d'àrees a fitxers CSV, només s'extreuen els atributs no espacials. Quan s'exporten entitats de punts, s'extreuen tant els atributs espacials com els no espacials.

Les dades extretes utilitzaran la projecció del Web Mercator 1984. Per als fitxers CSV, els camps Lat i Long s'annexaran a les dades amb coordenades al Web Mercator 1984.

S'admeten els formats de sortida següents:

  • Valors separats per comes (CSV), .csv o .zip
  • Geobase de dades de fitxers: .zip
  • Keyhole Markup Language (KML), .kmz o .zip
  • Shapefile: .zip

Nom del fitxer de sortida

És el nom de l'element que s'afegirà a El meu contingut. El nom de l'element es basa en el Format de les dades de sortida que s'ha triat. Si l'element ja existeix, se us demanarà que escriviu un altre nom.

Mitjançant el quadre desplegable Desa el resultat a, podeu especificar el nom d'una carpeta d' El meu contingut on es desarà el resultat.
