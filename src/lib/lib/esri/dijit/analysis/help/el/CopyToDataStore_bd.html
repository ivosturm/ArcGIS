Αντιγραφή στο Data Store

Το εργαλείο αντιγράφει ένα θεματικό επίπεδο εισόδου σε ένα data store. Μπορείτε να αντιγράφετε τα αποτελέσματα στο ArcGIS Data Store (datastore για σχεσιακά ή χωροχρονικά big data) οπότε θα δημιουργείται ένα θεματικό επίπεδο στο portal σας or μπορείτε να αντιγράφετε τα αποτελέσματα σε ένα datastore για αρχεία big data.

Για παράδειγμα, μπορείτε να αντιγράψετε στοιχεία που είναι αποθηκευμένα σε ένα κοινόχρηστο αρχείο μαζικών δεδομένων σε ένα σχεσιακό data store και να ορίσετε ότι θα αντιγράφονται μόνο τα στοιχεία εντός της τρέχουσας έκτασης χάρτη. Έτσι θα δημιουργούνταν ένα φιλοξενούμενο feature service μόνο με τα στοιχεία εντός της καθορισμένης έκτασης χάρτη.

Διαλέξτε θεματικό επίπεδο προς αντιγραφή

Το θεματικό επίπεδο ή ο πίνακας που θα αντιγραφεί.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να χρησιμοποιήσετε την Επιλογή θεματικού επιπέδου ανάλυσης στο κάτω μέρος της αναπτυσσόμενης λίστας για περιήγηση στο περιεχόμενό σας και αναζήτηση ενός κοινόχρηστου συνόλου δεδομένων αρχείων big data ή ενός feature layer.

Μπορείτε να χρησιμοποιήσετε όλα τα στοιχεία ή να επιλέξετε (προαιρετικό)

Προαιρετικά, μπορείτε να χρησιμοποιήσετε το κουμπί επιλογής για να ορίσετε τις τοποθεσίες προς αντιγραφή.

Επιλογή του datastore στο οποίο θα γίνει η αντιγραφή

Το data store στο οποίο θα αντιγραφεί το θεματικό επίπεδο. Η παράμετρος αυτή θα αντικαταστήσει την επιλογή στο πεδίο Περιβάλλοντα ανάλυσης. Οι επιλογές είναι οι εξής:

  • ArcGIS Data Store ArcGIS Data Store
    • Χωροχρονικά—Αυτή είναι η προεπιλεγμένη ρύθμιση.
    • Σχεσιακά
  • Πρότυπα εισόδου κοινόχρηστων αρχείων big data
    • Μια λίστα διαθέσιμων προτύπων κοινόχρηστων αρχείων big data με το μορφότυπο bigDataFileShareName-templateName. Πρέπει να παραμετροποιήσετε τα πρότυπα κοινόχρηστων αρχείων big data file πριν την εκτέλεση του εργαλείου.

Όνομα εξαγόμενου θεματικού επιπέδου

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί. Αν πραγματοποιείτε εγγραφή σε ένα ArcGIS Data Store, τα αποτελέσματά σας θα αποθηκευτούν στον φάκελο Το περιεχόμενό μου και θα προστεθούν στον χάρτη. Αν πραγματοποιείτε εγγραφή σε ένα κοινόχρηστο αρχείο big data, τα αποτελέσματά σας θα αποθηκευτούν στο κοινόχρηστο αρχείο big data και θα προστεθούν στη διακήρυξή του. Δεν θα προστεθούν στον χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.

Όποτε πραγματοποιείτε εγγραφή σε ArcGIS Data Store (datastore για σχεσιακά ή χωροχρονικά big data) χρησιμοποιώντας το πτυσσόμενο πλαίσιο Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε μέσα στον φάκελο Το περιεχόμενό μου το όνομα ενός άλλου φακέλου όπου θα αποθηκευτεί το αποτέλεσμα.
