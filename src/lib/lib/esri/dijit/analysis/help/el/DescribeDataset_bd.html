Περιγραφή Συνόλου Δεδομένων

Το εργαλείο «Περιγραφή Συνόλου Δεδομένων» παρέχει μια αναλυτικότερη εικόνα του συνόλου δεδομένων σας. Προτού εκτελέσετε ανάλυση μεγάλου όγκου δεδομένων, χρησιμοποιήστε το εργαλείο αυτό για να καθορίσετε ποια είναι τα δεδομένα εισόδου σας και πού βρίσκονται. Με βάση τις προεπιλεγμένες ρυθμίσεις, το αποτέλεσμα θα είναι μια συμβολοσειρά JSON που θα σκιαγραφεί τους βασικούς περιγραφείς του θεματικού επιπέδου εισόδου σας και ένας πίνακας που θα δίνει αναλυτικές πληροφορίες όσον αφορά τα συνοπτικά στατιστικά στοιχεία για κάθε πεδίο.

Προαιρετικά μπορείτε να δημιουργήσετε πρόσθετα feature layer εξόδου για περαιτέρω περιγραφή των δεδομένων σας. Οι επιλογές είναι ένα δείγμα θεματικού επιπέδου ή ένα θεματικό επίπεδο έκτασης. Μπορείτε να επιλέξετε την εξαγωγή ενός ή και των δύο.

Για παράδειγμα, ας υποθέσουμε ότι σας έχει δοθεί ένα κοινόχρηστο αρχείο big data το οποίο περιέχει 15 σύνολα δεδομένων. Κάθε σύνολο δεδομένων έχει 10 εκατομμύρια στοιχεία περιοχών που αναπαριστούν κτήρια και σπίτια σε διάφορες περιφέρειες της χώρας σας. Σας έχει ανατεθεί να αναλύσετε την περιφέρεια C, αλλά δεν ξέρετε ποιο σύνολο δεδομένων περιέχει τα στοιχεία αυτά. Για να εξακριβώσετε ποιο σύνολο δεδομένων πρέπει να χρησιμοποιήσετε, εκτελέστε το εργαλείο «Περιγραφή Συνόλου Δεδομένων» και επιλέξτε να δημιουργήσετε ένα θεματικό επίπεδο έκτασης για να διερευνήσετε ποιο σύνολο δεδομένων εμπίπτει στην περιοχή της μελέτης σας.

Ας πάμε σε ένα άλλο παράδειγμα υποθέτοντας ότι σας έχει ανατεθεί να ολοκληρώσετε μια ροή εργασιών ανάλυσης για μεγάλο όγκο δεδομένων. Θέλετε να επιχειρήσετε να ασχοληθείτε με τη ροή εργασιών, αλλά ίσως χρειαστείτε πολλές ώρες ή ημέρες για να καλύψετε όλο το σύνολο δεδομένων σας. Αντί να σπαταλήσετε χρόνο και πόρους για την εκτέλεση της πλήρους ανάλυσης, δημιουργήσετε ένα δείγμα θεματικού επιπέδου για να κάνετε μια αποτελεσματική δοκιμή της συνολικής ροής των εργασιών σας.

Επιλογή συνόλου δεδομένων για περιγραφή

Το θεματικό επίπεδο που περιέχει στοιχεία σημείων, γραμμών, περιοχών ή στηλών προς περιγραφή, σύνοψη και δειγματοληψία.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να χρησιμοποιήσετε την Επιλογή θεματικού επιπέδου ανάλυσης στο κάτω μέρος της αναπτυσσόμενης λίστας για περιήγηση στο περιεχόμενό σας και αναζήτηση ενός κοινόχρηστου συνόλου δεδομένων αρχείων big data ή ενός feature layer.

Κατανοήστε το σύνολο δεδομένων σας δημιουργώντας (προαιρετικά)

Εξαγάγετε πρόσθετα περιγραφικά θεματικά επίπεδα για να βελτιώσετε την κατανόηση των big data που έχετε στη διάθεσή σας. Μπορείτε να επιλέξετε να εξαγάγετε κανένα, ένα ή και τα δύο από τα εξής θεματικά επίπεδα:

  • Θεματικό επίπεδο έκτασης—Δημιουργεί ένα στοιχείο περιοχών που αναπαριστά την έκταση των οντοτήτων εισόδου ή της περιοχής ενδιαφέροντός σας.
  • Δείγμα θεματικού επιπέδου—Δημιουργεί ένα θεματικό επίπεδο υποσυνόλου δεδομένων το οποίο περιέχει ένα καθορισμένο αριθμό οντοτήτων εισόδου από το σύνολο δεδομένων ή την περιοχή ενδιαφέροντός σας.

Δείγμα θεματικού επιπέδου

Εξαγάγετε ένα θεματικό επίπεδο που θα περιέχει ένα υποσύνολο στοιχείων από το θεματικό επίπεδο εισόδου σας. Αν επιλέξετε το Δείγμα θεματικού επιπέδου, έχετε τη δυνατότητα να καθορίσετε τον αριθμό των στοιχείων που θέλετε να επιστραφούν στο δείγμα θεματικού επιπέδου. Με βάση τις προεπιλεγμένες ρυθμίσεις, στο θεματικό επίπεδο εισόδου θα επιστραφούν 100 δειγματοληπτικά στοιχεία.

Η τιμή πρέπει να είναι μεγαλύτερη από το μηδέν. Αν καθορίσετε έναν αριθμό μεγαλύτερο από τον συνολικό αριθμό των στοιχείων, θα επιστραφούν όλα τα στοιχεία.

Αν επιλέξετε τη Χρήση τρέχουσας έκτασης χάρτη, τότε το δείγμα θεματικού επιπέδου θα περιέχει στοιχεία που εμπίπτουν στην έκταση του χάρτη.

Θεματικό επίπεδο έκτασης

Εξαγάγετε ένα feature layer που θα περιέχει ένα μεμονωμένο στοιχείο που αναπαριστά την έκταση των οντοτήτων εισόδου επιλέγοντας το κουμπί Θεματικό επίπεδο έκτασης. Το κουμπί αυτό είναι επιλεγμένο με βάση τις προεπιλεγμένες ρυθμίσεις.

Αν επιλέξετε τη Χρήση τρέχουσας έκτασης χάρτη, τότε το θεματικό επίπεδο έκτασης θα αναπαριστά την ορατή έκταση του χάρτη.

Όνομα εξαγόμενου θεματικού επιπέδου

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί. Αν πραγματοποιείτε εγγραφή σε ένα ArcGIS Data Store, τα αποτελέσματά σας θα αποθηκευτούν στον φάκελο Το περιεχόμενό μου και θα προστεθούν στον χάρτη. Αν πραγματοποιείτε εγγραφή σε ένα κοινόχρηστο αρχείο big data, τα αποτελέσματά σας θα αποθηκευτούν στο κοινόχρηστο αρχείο big data και θα προστεθούν στη διακήρυξή του. Δεν θα προστεθούν στον χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.
