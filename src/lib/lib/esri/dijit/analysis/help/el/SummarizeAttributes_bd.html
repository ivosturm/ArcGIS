Σύνοψη Γνωρισμάτων

Το εργαλείο αυτό συνοψίζει όλες τις τιμές που συμφωνούν σε ένα ή περισσότερα πεδία και υπολογίζει στατιστικά για τις εν λόγω τιμές. Το βασικότερο στατιστικό είναι το πλήθος των στοιχείων που έχουν συνοψιστεί μαζί, αλλά μπορείτε να υπολογίσετε και άλλα πιο σύνθετα στατιστικά στοιχεία.

Για παράδειγμα, ας υποθέσουμε ότι έχετε σημειακά στοιχεία τοποθεσιών καταστημάτων με ένα πεδίο που αντιπροσωπεύει το DISTRICT_MANAGER_NAME, και θέλετε να συνοψίσετε τις πωλήσεις καφέ κατά διευθυντή. Μπορείτε να προσδιορίσετε το πεδίο DISTRICT_MANAGER_NAME ως το πεδίο για ομαδοποίηση, και θα συνοψιστούν όλες οι σειρές δεδομένων που αντιστοιχούν σε μεμονωμένους διευθυντές. Αυτό σημαίνει ότι όλες οι τοποθεσίες καταστημάτων τις οποίες διαχειρίζεται ο 

Manager1
θα συνοψιστούν σε μία σειρά, με τα συνοψισμένα στατιστικά στοιχεία υπολογισμένα. Στην περίπτωση αυτή θα υπολογιζόντουσαν στατιστικά στοιχεία όπως ο αριθμός των καταστημάτων και το άθροισμα όσον αφορά TOTAL_SALES για όλα τα καταστήματα που διαχειρίζεται ο 
Manager1
, καθώς και κάθε άλλος διευθυντής που παρατίθεται στο πεδίο DISTRICT_MANAGER_NAME.

Διαλέξτε τα δεδομένα εισαγωγής προς σύνοψη

Το θεματικό επίπεδο που περιέχει τα πεδία που θα συνοψιστούν.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να χρησιμοποιήσετε την Επιλογή θεματικού επιπέδου ανάλυσης στο κάτω μέρος της αναπτυσσόμενης λίστας για περιήγηση στο περιεχόμενό σας και αναζήτηση ενός κοινόχρηστου συνόλου δεδομένων αρχείων big data ή ενός feature layer.

Επιλογή τρόπου συνόψισης των δεδομένων σας

Υπάρχουν δύο τρόποι συνόψισης των δεδομένων σας:

  • Όλα τα στοιχεία σε ένα ενιαίο στοιχείο— Υπολογίζονται στατιστικά για όλα τα στοιχεία.
  • Κατά πεδία— Ομαδοποιούνται στοιχεία με βάση παρεμφερείς τιμές σε καθορισμένα πεδία.

Για παράδειγμα, ας υποθέσουμε ότι έχετε ένα σύνολο δεδομένων δέντρων που περιλαμβάνουν πεδία για treetype με τιμές Maple, Fir και Pine και ένα πεδίο για treeheight και σας ενδιαφέρει να βρείτε τον μέσο όρο της τιμής treeheight. Αν συνοψίσετε όλα τα στοιχεία σε ένα ενιαίο στοιχείο, θα βρείτε το μέσο ύψος όλων των δέντρων, δηλαδή μία μέση τιμή. Αν συνοψίζατε με βάση το πεδίο treeheight, θα καταλήγατε να έχετε μια μέση τιμή για τους σφένδαμους, μια μέση τιμή για τα έλατα και μια μέση τιμή για τα πεύκα.

Προσθήκη στατιστικών (προαιρετικά)

Μπορείτε να υπολογίσετε στατιστικά για τα στοιχεία που συνοψίζονται.Μπορείτε να υπολογίσετε τα στοιχεία που ακολουθούν στα αριθμικά πεδία:

  • Αριθμός—Υπολογίζει τον αριθμό τιμών που δεν είναι μηδενικές (null). Μπορεί να χρησιμοποιηθεί σε αριθμικά πεδία ή σε συμβολοσειρές. Το πλήθος των [null, 0, 2] είναι 2.
  • Άθροισμα—Το άθροισμα των αριθμητικών τιμών σε ένα πεδίο. Το άθροισμα των [null, null, 3] είναι 3.
  • Μέση τιμή—Η μέση τιμή αριθμητικών τιμών. Η μέση τιμή των [0, 2, null] είναι 1.
  • Ελάχιστο—Η ελάχιστη τιμή ενός αριθμητικού πεδίου. Το ελάχιστο των [0, 2, null] είναι 0.
  • Μέγιστο—Η μέγιστη τιμή ενός αριθμητικού πεδίου. Η μέγιστη τιμή των [0, 2, null] είναι 2.
  • Εύρος—Το εύρος ενός αριθμητικού πεδίου. Αυτό υπολογίζεται ως οι ελάχιστες τιμές που αφαιρούνται από την μέγιστη τιμή. Το εύρος των [0, null, 1] είναι 1. Το εύρος των [null, 4] είναι 0.
  • Διακύμανση—Η διακύμανση ενός αριθμητικού πεδίου σε ένα ίχνος. Η διακύμανση του [1] είναι null. Η διακύμανση των τιμών [null, 1,0,1,1] είναι 0,25.
  • Τυπική απόκλιση—Η τυπική απόκλιση ενός αριθμητικού πεδίου. Η τυπική απόκλιση του [1] είναι null. Η τυπική απόκλιση των τιμών [null, 1,0,1,1] είναι 0,5.

Μπορείτε να υπολογίσετε τα στοιχεία που ακολουθούν στα πεδία συμβολοσειρών:

  • Πλήθος—Ο αριθμός των συμβολοσειρών που δεν είναι null.
  • Οποιαδήποτε—Αυτό το στατιστικό στοιχείο είναι ένα τυχαίο δείγμα μιας τιμής συμβολοσειράς στο καθορισμένο πεδίο.
Όλα τα στατιστικά στοιχεία υπολογίζονται σε τιμές που δεν είναι null. Το εξαγόμενο θεματικό επίπεδο θα περιέχει ένα νέο πεδίο για κάθε υπολογιζόμενο στατιστικό στοιχείο. Μπορεί να προστεθεί οποιοσδήποτε αριθμός στατιστικών στοιχείων, επιλέγοντας γνώρισμα και στατιστικό στοιχείο.

Επιλογή χώρου αποθήκευσης δεδομένων

Τα αποτελέσματα GeoAnalytics αποθηκεύονται σε ένα datastore και εκτίθενται ως feature layer στο Portal for ArcGIS. Στις περισσότερες περιπτώσεις, τα αποτελέσματα λογικά θα αποθηκεύονται στο datastore για χωροχρονικά δεδομένα και αυτή είναι η προεπιλεγμένη ρύθμιση. Σε ορισμένες περιπτώσεις, μια καλή επιλογή είναι η αποθήκευση αποτελεσμάτων στο σχεσιακό data store. Ακολουθούν ορισμένοι λόγοι για τους οποίους θα θέλατε να αποθηκεύσετε αποτελέσματα στο σχεσιακό data store:

  • Μπορείτε να χρησιμοποιήσετε αποτελέσματα σε συνεργασία portal με portal.
  • Μπορείτε να ενεργοποιήσετε τις δυνατότητες συγχρονισμού με τα αποτελέσματά σας.

Όνομα εξαγόμενου θεματικού επιπέδου

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί. Αν πραγματοποιείτε εγγραφή σε ένα ArcGIS Data Store, τα αποτελέσματά σας θα αποθηκευτούν στον φάκελο Το περιεχόμενό μου και θα προστεθούν στον χάρτη. Αν πραγματοποιείτε εγγραφή σε ένα κοινόχρηστο αρχείο big data, τα αποτελέσματά σας θα αποθηκευτούν στο κοινόχρηστο αρχείο big data και θα προστεθούν στη διακήρυξή του. Δεν θα προστεθούν στον χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.

Όποτε πραγματοποιείτε εγγραφή σε ArcGIS Data Store (datastore για σχεσιακά ή χωροχρονικά big data) χρησιμοποιώντας το πτυσσόμενο πλαίσιο Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε μέσα στον φάκελο Το περιεχόμενό μου το όνομα ενός άλλου φακέλου όπου θα αποθηκευτεί το αποτέλεσμα.
