GeoAnalytics se centra en analizar grandes cantidades de datos incidiendo en los patrones espaciales y temporales.

Summarize Data

The Summarize Data toolset contains tools that calculate total counts, lengths, areas, and basic descriptive statistics of features and their attributes within areas or near other features.

  • Aggregate Points calculates statistics about points that fall within specified areas or bins.
  • Build Multi-Variable Grid generates a grid of square or hexagonal bins and calculates values for each bin based on the proximity of each input layer.
  • Describe Dataset calculates a summary of your input, and can optionally generate a sample of your input layer.
  • Join Features calculates statistics about features that share a spatial, temporal, or attribute relationship with other features.
  • Reconstruct Tracks calculates statistics about points or polygons that belong to the same track and reconstructs inputs into tracks.
  • Summarize Attributes calculates statistics about feature or tabular data that share attributes.
  • Summarize Within calculates statistics for area features and attributes that overlap each other.

Consolidar puntos

Utilizando una capa de entidades de puntos y una capa de entidades de área o bins definida mediante una distancia especificada, esta herramienta determina qué puntos quedan dentro de cada área o bin y calcula las estadísticas de todos los puntos dentro de cada área o bin. Si lo desea, puede aplicar la función de intervalo de tiempo con esta herramienta.

A continuación se muestran algunos ejemplos:

  • Dadas las ubicaciones de puntos de los incidentes criminales, cuenta el número de crímenes por condado o por otros distritos administrativos.
  • Busca los ingresos mensuales más altos y más bajos para las ubicaciones de franquicia utilizando bins de 100 km.


Crear cuadr√≠cula multivariable

La herramienta Crear cuadrícula multivariable genera una cuadrícula de bins cuadrados o hexagonales y calcula variables para cada bin basándose en la proximidad de una o varias capas de entrada.

A continuación se muestran algunos ejemplos:

  • Considerando varias capas de infraestructuras de transporte público, ¿qué parte de la ciudad es la menos accesible mediante transporte público?
  • Considerando capas de vías acuáticas, tales como lagos y ríos, ¿cuál es el nombre de la masa de agua más cercana a cada ubicación en los EE. UU.?
  • Considerando una capa de ingresos por hogar, ¿en qué parte de los EE. UU. es mayor la variación entre los ingresos de 50 millas a la redonda?


Describir dataset

Resume entidades de entrada con estadísticas, capas de muestra y visualización. Puede elegir que se genere una capa de muestra o una capa de extensión.

A continuación se muestran algunos ejemplos:

  • Hace referencia a estadísticas de resumen calculadas para comprender los valores que contiene su dataset.
  • En lugar de dibujar todo el dataset, crea una capa de subconjunto para visualizar entidades y sus atributos cuando se agrega a un mapa.
  • Le permite ahorrar tiempo y recursos, y ejecuta análisis en una capa de muestra antes de ejecutarlos en el dataset completo.


Unir entidades

Utilizando capas de entidades o datos tabulares, puede unir entidades y registros basándose en relaciones específicas entre las capas o tablas de entrada. Las uniones vendrán determinadas por las relaciones espaciales, temporales y de atributos, y también es posible calcular resumen de estadísticas.

A continuación se muestran algunos ejemplos:

  • Dadas unas ubicaciones de puntos de incidentes delictivos en un intervalo tiempo, une a sí mismo los datos de los delitos especificando una relación espacial de los delitos cometidos en un radio de 1 kilómetro de distancia y producidos con una diferencia de 1 hora entre sí para determinar si existe una secuencia de delitos cercanos entre sí en espacio y tiempo.
  • Dada una tabla de códigos postales con información demográfica y entidades de área que representan edificios residenciales, une la información demográfica a las viviendas para que cada vivienda ahora disponga de la información.


Reconstruir recorridos

Utilizando una capa de entidades de puntos o poligonales con la función de tiempo habilitada que representa un instante en el tiempo, esta herramienta determina qué entidades de entrada pertenecen a un recorrido y ordena las entradas secuencialmente en el tiempo. Opcionalmente, se calculan estadísticas para las entidades de entrada dentro de cada recorrido.

A continuación se muestran algunos ejemplos:

  • Dadas unas ubicaciones de puntos y mediciones sobre el momento en que se produjo el huracán, calcula la velocidad media del viento y la presión máxima del viento del huracán.
  • Utiliza mediciones GPS de barcos para reconstruir las rutas de los barcos como líneas.


Resumir atributos

Utilizando datos de entidades o tabulares, esta herramienta resume estadísticas para campos específicos.

A continuación se muestran algunos ejemplos:

  • Dadas unas ubicaciones de tiendas de comestibles con el campo COMPANY_NAME, resume las tiendas por el nombre de la empresa para determinar estadísticas para cada empresa.
  • Dada una tabla de tiendas de comestibles con los campos COMPANY_NAME y COUNTY, resume las tiendas por el nombre de la empresa y condado a fin de determinar estadísticas para cada empresa dentro de cada condado.


Resumir dentro de

Busca áreas (y porciones de áreas) que se superponen entre dos capas y calcula estadísticas sobre la superposición.

A continuación se muestran algunos ejemplos:

  • Dada una capa de áreas de cuencas hidrográficas y una capa de áreas de uso del suelo, calcule la superficie total del tipo de uso del suelo para cada cuenca hidrográfica.
  • Dada una capa de parcelas de un condado y una capa de límites de ciudades, resuma el valor medio de las parcelas vacías en cada ciudad.


Buscar ubicaciones

Estas herramientas se utilizan para identificar las áreas que cumplen diversos criterios especificados.

  • Detectar incidentes trabaja con capas con la función de tiempo habilitada para determinar las entidades que cumplen una condición especificada.
  • Geocodificar ubicaciones de tabla convierte direcciones en coordenadas.
  • Buscar ubicaciones similares busca las ubicaciones más similares a una o varias ubicaciones de referencia en función de los criterios especificados.

Detectar incidentes

Esta herramienta funciona con una capa de puntos, líneas, áreas o tablas con la función de tiempo habilitada que representa un instante en el tiempo. Si utiliza entidades ordenadas secuencialmente, denominadas recorridos, esta herramienta determina qué entidades son incidentes de interés. Los incidentes se determinan atendiendo a las condiciones que usted especifique.

A continuación se muestran algunos ejemplos:

  • Detecte mediciones de huracanes de categoría 4 en función de un atributo de velocidad del viento.
  • Con las mediciones GPS del movimiento del barco, con un campo que represente la velocidad del mismo, busque mediciones 1,5 veces mayores que el valor medio de las cinco mediciones anteriores.


Geocodificar ubicaciones de tabla

Convierte direcciones en coordenadas. Utilice esta herramienta en tablas de uso compartido de archivos de big data.


Buscar ubicaciones similares

En función de los criterios que se especifiquen, puede buscar ubicaciones similares midiendo la similitud de las ubicaciones de la capa de búsqueda de candidatos con una o varias ubicaciones de referencia.

A continuación se muestran algunos ejemplos:

  • Busque las diez tiendas más similares examinando la cantidad de empleados y las ventas anuales.
  • Busque las 100 ciudades más similares examinando la relación entre la población, el crecimiento anual y los ingresos derivados de los impuestos.


Analizar patrones

Estas herramientas le ayudan a identificar, cuantificar y visualizar los patrones espaciales de sus datos.

  • Calcular densidad toma cantidades conocidas de un fenómeno y las expande a través del mapa.
  • Buscar puntos calientes identifica clustering estadísticamente significativo en el patrón espacial de sus datos.
  • Buscar clústeres de puntos encuentra clústeres de entidades de punto dentro del ruido colindante en función de su distribución espacial.
  • Clasificación y regresión basadas en bosque crea modelos y genera predicciones mediante una adaptación del algoritmo de bosque aleatorio de Leo Breiman, que es un método de aprendizaje de máquina supervisado.
  • Regresión lineal generalizada genera predicciones o modela una variable dependiente en términos de su relación con un conjunto de variables explicativas.

Calcular densidad

La herramienta Calcular densidad crea un mapa de densidad a partir de entidades de punto mediante la extensión de cantidades conocidas de algún fenómeno (representado como atributos de los puntos) a todo el mapa. El resultado es una capa de áreas que representan la densidad.

A continuación se muestran algunos ejemplos:

  • Calcular las densidades de hospitales en un condado. La capa de resultados mostrará las áreas con una accesibilidad alta y baja a los hospitales, y esta información se puede usar para decidir dónde se deben construir nuevos hospitales.
  • Identificar áreas con un alto riesgo de incendio forestal en función del historial de ubicaciones de estos incendios.
  • Buscar comunidades alejadas de las principales carreteras para planificar dónde se deben construir nuevas carreteras.

A continuación se muestran algunos ejemplos:

  • Informe y actúe sobre las necesidades de rescate y evacuación en función del tamaño y la ubicación de los clústeres por medio de tuits geolocalizados después de una catástrofe natural o un ataque terrorista.


Buscar puntos calientes

La herramienta Buscar puntos calientes determinará si hay algún clustering estadísticamente significativo en el patrón espacial de sus datos.

  • ¿Están realmente agrupados sus puntos (incidentes criminales, árboles, accidentes de tráfico)? ¿Cómo puede estar seguro?
  • ¿Ha descubierto de verdad un punto caliente estadísticamente significativo (para el gasto, la mortalidad infantil, notas de examen sistemáticamente altas) o contaría su mapa una historia diferente si cambiara sus símbolos?
La herramienta Buscar puntos calientes le ayudará a responder a estas preguntas con confianza.

Crear zonas de influencia

Una zona de influencia es un área que cubre una distancia dada desde una entidad de puntos, de línea o de entidad poligonal.

A continuación se muestran algunos ejemplos:

  • Utilizando entidades de río lineales, crea una zona de influencia de cada río multiplicando por 50 el ancho del río para determinar el límite ribereño propuesto.
  • Dadas áreas que representan países, crea una zona de influencia de cada país por 200 millas náuticas para determinar el límite marítimo.


Manage Data

These tools are used for the day-to-day management of geographic and tabular data.

  • Append Data appends features to an existing hosted feature layer.
  • Calculate Field creates a new field or modifies an existing field and creates a new layer in your contents.
  • Clip Layer copies features that intersect a defined area of interest.
  • Copy to Data Store copies data to your ArcGIS Data Store and creates a layer in your contents.
  • Dissolve Boundaries merges areas that share a common boundary or attribute value.
  • Merge Layers copies all features from two layers into a single result layer.
  • Overlay Layers combines two or more layers into a single layer.

Superponer capas

Combina dos o más capas en una sola capa. La superposición se utiliza para responder a una de las preguntas geográficas más sencillas: ¿Qué está encima de qué?

A continuación se muestran algunos ejemplos:

  • ¿Qué pozos están dentro de bases militares abandonadas?

Fusionar capas

Combina dos capas de entidades para crear una única capa de resultados. Todos los campos de la capa de fusión se incluirán de forma predeterminada; también puede especificar reglas de fusión personalizadas para definir el esquema resultante.

A continuación se muestran algunos ejemplos:

  • Fusionar tres capas que contienen información demográfica de Inglaterra, Gales y Escocia para crear una única capa de Gran Bretaña.
  • Combinar dos capas que contienen información de parcelas de municipios contiguos, conservando únicamente los campos que tengan el mismo nombre y tipo en ambas capas.
