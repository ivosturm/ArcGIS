Arvuta vahemaa

See tööriist arvutab Eukleidese kauguse, suuna ja paigutuse ühestainsast lähtepunktist või lähtepunktide komplektist.

Näidisrakendused, sealhulgas alljärgnevatele küsimustele vastamine:

  • Milline on vahemaa lähima linnani?
  • Milline on vahemaa lähima maanteeni?
  • Milline on vahemaa lähima veekoguni?

Kui ruut Kasuta praegust kaardi ulatust on märgitud, analüüsitakse ainult neid sisendkihi objekte, mis on nähtavad kaardi praeguses ulatuses. Kui see ruut pole märgitud, kasutatakse analüüsi koostamisel kõiki sisendkihis olevaid objekte ka siis, kui neid pole praegu kaardil näha.

Vahemaa arvutamiseks valige objekti asukohakihi raster

Kiht, mis määrab lähtepunktid, milleni vahemaa arvutatakse. Kiht võib olla pilditeenus või objektiteenus.

Pilditeenuse puhul võib sisendtüüp olla täisarv või ujuvpunkt.

Objektiteenuse puhul võib sisend olla punkt, joon või hulknurk.

Maksimaalne arvutatav vahemaa

Maksimaalne vahemaa, mis tuleb arvutada. Kui esitate näiteks väärtuse 10 000 meetrit, arvutatakse Eukleidese kaugus lähtepunktidest kuni selle väärtuseni (kaugus).

Kui väärtust ei esitata, arvutatakse kaugus kuni määratletud ulatuseni.

Kasutatavad ühikud on kilomeeter, meeter, miil ja jalg.

Väljundpiksli suurus

The cell size and unit for the output rasters.

The units can be kilometers, meters, miles, or feet.

Tulemkauguse kihi nimi

Selle kihi nimi, mis luuakse jaotises Minu sisu ja lisatakse kaardile. Vaikimisi nimi põhineb tööriista nimel ja sisendkihi nimel. Kui kiht on juba olemas, palutakse teil sisestada mõni muu nimi.

You can specify the name of a folder in My Content where the result will be saved using the Save result in drop-down box.

Tulemsuuna kihi nimi

Jaotises Minu sisuloodava kihi nimi. You must provide a name that does not currently exist in My Content.

You can specify the name of a folder in My Content where the result will be saved using the Save result in drop-down box.

Vali paigutusväli

Väli sisendallika kihis, mis sisaldavad igat allikat määratlevaid väärtusi. See võib olla täisarv või lähte-andmekogumi stringiväli.

Vaikeväärtus on Value. Kui kasutatakse objektiteenust ja väärtuse Value väli ei ole saadaval, siis kasutatakse esimest saadaolevat täisarvu või stringi välja. Kui kehtivaid välju ei ole, kasutatakse välja objekti ID-d (objekti sisendi tüübist sõltuvalt näiteks OID või FID).
