Salin ke Penyimpanan Data

Alat ini akan mengambil layer input dan menyalinnya ke penyimpanan data. Anda dapat menyalin hasil ke ArcGIS Data Store (penyimpanan big data relasional atau spasiotemporal) dan sebuah layer akan dibuat di portal Anda atau Anda dapat menyalin hasil ke sebuah file big data.

Contoh, Anda dapat menyalin fitur yang disimpan dalam berbagi file big data ke penyimpanan data relasional dan menentukan bahwa hanya fitur dalam jangkauan peta saat ini yang akan disalin. Hal ini akan membuat feature service yang di-host hanya dengan fitur yang berada di dalam jangkauan peta yang telah ditentukan.

Pilih layer untuk disalin

Layer atau tabel yang akan disalin.

Selain memilih layer dari peta, Anda dapat memilih Pilih Layer Analisis di bawah daftar drop-down untuk menelusuri konten Anda untuk set data atau feature layer berbagi file big data.

Anda dapat menggunakan semua fitur atau membuat pilihan (opsional)

Atau, gunakan tombol pemilih untuk mengidentifikasi lokasi yang akan disalin.

Pilih penyimpanan data untuk disalin

Penyimpanan data yang mana layernya akan disalin. Parameter ini akan menimpa pilihan dalam Lingkungan Analisis. Opsinya adalah sebagai berikut:

  • Penyimpanan Data ArcGIS ArcGIS Data Store
    • Spasiotemporal—Ini adalah default.
    • Relasional
  • Template Output Berbagi File Big Data
    • Sebuah daftar template berbagi file big data terdapat dalam format bigDataFileShareName-templateName. Template berbagi file big data harus dikonfigurasi sebelum menjalankan alat.

Nama layer hasil

Nama layer yang akan dibuat. Jika Anda menulis ke sebuah ArcGIS Data Store, hasil Anda akan disimpan di Konten Saya dan ditambahkan ke dalam peta. Jika Anda menulis ke berbagi file big data, hasil Anda akan disimpan ke berbagi file big data dan ditambahkan ke manifestasinya. Ini tidak akan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer sudah ada, alat tidak akan berfungsi.

Saat menulis ke ArcGIS Data Store (penyimpanan big data relasional atau spasiotemporal) menggunakan Simpan hasil di kotak drop-down, Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan.
