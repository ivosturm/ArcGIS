Geocode Lokasi dari Tabel

Geocoding mengonversi alamat menjadi koordinat. Alat ini mengubah alamat dari tabel berbagi file big data menjadi feature layer dengan koordinat sebagai kolom tambahan.

Pilih tabel input

Tabel input dengan alamat akan di-geocode.

Pilih penentu lokasi

Pilih service penentu lokasi untuk melakukan geocode. Jika service penentu lokasi yang ingin Anda gunakan tidak berada di menu tarik-turun, hubungi administrator agar service penentu lokasi dikonfigurasi untuk geocoding kelompok.

Negara

Jika semua alamat Anda berasal dari satu negara, pilih negara tersebut untuk memastikan hasil yang paling akurat. Jika tidak, kosongkan parameter ini.

Pilih Kolom Data

Layanan penentu lokasi membutuhkan kolom tertentu di data, seperti alamat, kota, negara bagian, dan kode pos. Cocokkan kolom data di file input atau tabel dengan kolom yang diharapkan penentu lokasi dengan menggunakan kotak drop-down. Semakin banyak kolom yang Anda petakan, hasilnya akan semakin akurat. Jika semua data tabel input berada di satu kolom, pilih Kolom Tunggal, dan pilih kolom data dari menu drop-down. Jika tidak, pilih Beberapa Kolom, dan petakan kolom penentu lokasi yang sesuai dengan kolom dengan banyak bidang di tabel.

Untuk setiap lokasi

Sertakan kolom tambahan yang dikembalikan oleh layanan lokasi. Jika opsi ini dicentang, semua kolom yang dibuat oleh penentu lokasi akan dikembalikan. Jika tidak dicentang, data input Anda akan dikembalikan dengan koordinat yang di-geocode.

Nama hasil output

Nama layer yang akan dibuat. Jika Anda menulis ke sebuah ArcGIS Data Store, hasil Anda akan disimpan di Konten Saya dan ditambahkan ke dalam peta. Jika Anda menulis ke berbagi file big data, hasil Anda akan disimpan ke berbagi file big data dan ditambahkan ke manifestasinya. Ini tidak akan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer sudah ada, alat tidak akan berfungsi.

Saat menulis ke ArcGIS Data Store (penyimpanan big data relasional atau spasiotemporal) menggunakan Simpan hasil di kotak drop-down, Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan.
