Aggiungi dati da griglia multi-variabile

Lo strumento Aggiungi dati da griglia multi-variabile unisce efficacemente gli attributi da una griglia multi-variabile a un layer puntuale.

Una griglia multi-variabile è un layer creato con lo strumento Crea griglia multi-variabile, che compila attributi da vari layer in un'unica griglia di poligoni. L'uso di una griglia multi-variabile per l'aggiunta di dati consente di aggiungere rapidamente raccolte grandi e variegate di informazioni ai dati puntuali da usare in ulteriori analisi spaziali.

Ad esempio, immaginiamo di aver ottenuto un layer di nuovi eventi criminali verificatisi l'anno scorso in una città. Si desidera modellare le relazioni tra la frequenza dei crimini e una varietà di fattori demografici, economici e geografici. Si ha accesso a dati che descrivono questi fattori, ma tali dati sono memorizzati in cinque layer diversi con tipi di geometria diversi. Inoltre, alcuni layer sono memorizzati in una condivisione file Big Data, mentre altri sono memorizzati come feature service. Creando una griglia multi-variabile per memorizzare queste informazioni e unendo quindi la griglia ai propri eventi criminali con lo strumento Aggiungi dati da griglia multi-variabile, è possibile unire in modo efficace una grande varietà di attributi al proprio layer puntuale con l'avvio di solo due strumenti. È quindi possibile usare gli attributi uniti come variabili esplicative e iniziare a costruire un modello di frequenza dei crimini con il layer dei risultati.

Scegliere le feature puntuali cui aggiungere dati

Le feature puntuali nelle quali saranno aggiunti dati dalla griglia multi-variabile.

Oltre a scegliere un layer dalla mappa, è possibile selezionare Scegli layer di analisi alla base dell'elenco a discesa per cercare un dataset di condivisione file Big Data o un feature layer.

Scegliere la griglia multi-variabile a partire dalla quale aggiungere dati

Il layer della griglia multi-variabile contenente attributi che saranno uniti alle feature puntuali di input. Il layer deve essere stato creato usando lo strumento Crea griglia multi-variabile e deve essere un feature layer hosted. I layer della griglia multi-variabile memorizzati in una condivisione file Big Data non sono supportati come input per questo strumento.

Scegliere le variabili da aggiungere

Gli attributi nel layer della griglia multi-variabile che saranno uniti alle feature puntuali di input quando le feature puntuali intersecano la griglia.

Nome del layer dei risultati

Il nome del layer che verrà creato. Se si scrive su un ArcGIS Data Store, i risultati saranno salvati in I miei contenuti e aggiunti alla mappa. Se si scrive su una condivisione file Big Data, i risultati saranno memorizzati nella condivisione file Big Data e aggiunti al suo Manifest. Non saranno aggiunti alla mappa. Il nome predefinito è basato sul nome dello strumento e sul nome del layer di input. Se il layer esiste già, lo strumento non verrà eseguito.
