데이터 추가

이 도구는 기존 호스팅 피처 레이어에 데이터를 추가합니다. 데이터 추가는 기존 입력 레이어를 수정하고 새로운 결과 레이어를 생성하지 않습니다. 필드 이름 및 필드 유형을 기반으로 필드를 일치하거나 고급 일치 방법을 적용할 수 있습니다.

예를 들면 다음과 같습니다.

  • 월간 판매 기록을 생성하는 대형 보험 회사의 직원이라고 가정해 보겠습니다. 연간 보고서의 경우 분석을 위해 월간 데이터를 수집해야 합니다. 판매 분석을 시작하려면 월간 데이터 사용이 가능할 때 데이터 추가를 활용하여 AnnualReport_2018 레이어를 업데이트할 수 있습니다. 매월 기관에서 일관된 스키마를 유지한다는 것을 알고 있으므로, 기본 필드 매핑 방법을 사용하여 필드 이름 및 유형을 기반으로 피처를 효율적으로 추가할 수 있습니다.
  • 10개의 환경 단체는 각 관할 구역 내에서 수십만 개의 기후 측정값을 수집합니다. 이러한 기관은 지역 및 글로벌 패턴에 대한 이해를 돕기 위해 데이터를 수집하고자 합니다. 각 기관은 해당 데이터셋의 스키마를 결정합니다. 기관은 온도와 풍속 데이터에 각각 다른 필드 이름을 지정했습니다. 데이터셋 간의 필드 이름에 차이가 있는 경우, 필드 매핑을 활용하여 온도와 풍속 측정값을 개별 필드에 추가하는 방법을 지정할 수 있습니다.

다음에 추가할 레이어 선택

데이터가 추가될 테이블, 포인트, 라인 또는 폴리곤 피처입니다.

Caution: 

추가할 레이어는 도구를 실행하는 동안 수정됩니다. 진행하기 전에 이 레이어를 수정할 것인지 확인합니다.

맵에서 레이어를 선택하는 것 이외에, 드롭다운 목록의 하단에 있는 Analysis Layer 선택을 선택하여 호스팅 피처 레이어의 콘텐츠를 찾아볼 수 있습니다.

추가할 레이어 선택

입력 레이어에 추가할 테이블, 포인트, 라인 또는 폴리곤 피처입니다.

맵에서 레이어를 선택하는 것 이외에, 드롭다운 목록의 하단에 있는 Analysis Layer 선택을 선택하여 빅데이터 파일 공유 데이터셋 또는 피처 레이어의 콘텐츠를 찾아볼 수 있습니다.

모든 피처 추가 또는 일부분 정의(선택)

추가할 피처를 식별합니다. 매개변수는 선택 사항입니다.

필드 추가 방법 선택(선택)

추가 레이어 필드를 입력 레이어에 추가하는 방법을 지정합니다. 기본 설정에 따라 일치하는 모든 입력 및 추가 필드가 추가됩니다. 필드 이름이 같고 필드 유형이 동일한 경우 필드는 일치합니다. 필요한 경우 필드가 일치되는 방법을 선택하거나 Arcade 식을 계산하여 필드 값을 채울 수 있습니다.

예를 들어, 입력 레이어에 field_number, field_string, date_field및 unused_field 필드가 있고 추가 레이어에 field_string, field_number 및 MY_DATE_FIELD 필드가 있습니다.

  • 기본 설정에 따라, 도구는 입력 레이어의 field_number를 추가 레이어의 field_number와 일치시키고 입력 레이어의 field_string을 추가 레이어의 field_string와 일치시킵니다.
  • 기본 설정에 따라, 입력 레이어에는 추가 레이어에 일치하는 필드가 없으므로 입력 레이어 date_field 및 unused_field 필드는 null 값으로 채워집니다.
  • 선택적으로 기본 설정에 따라 자동으로 일치하지 않는 필드에 값을 추가할 수 있습니다. 예를 들어, 입력 레이어의 date_field는 추가 필드 MY_DATE_FIELD와 일치할 수 없습니다. 그렇게 하려면 첫 번째 열의 date_field 옆에 있는 드롭다운 메뉴를 사용하고 두 번째 열의 드롭다운 메뉴에서 MY_DATE_FIELD를 선택합니다.
또 다른 예시로, 입력 레이어에 value_1, value_2, value_3 및 my_Average 숫자 필드가 있고 추가 레이어에 value_1, value_2 및 value_3 숫자 필드가 있습니다.
  • 기본 설정에 따라, value_1, value_2 및 value_3 값은 일치하고 추가됩니다.
  • 기본 동작 외에도 my_Average 필드의 값을 계산할 수 있습니다. my_Average 옆에 있는 열의 식 드롭다운 옵션을 선택하고 
    average($feature["value_1"] + $feature["value_2"] + $feature["value_3"])
    와 같은 사용자 정의 Arcade 식을 생성하는 등 다중 필드 및 계산 작업을 사용하여 이 작업을 수행합니다.
