O GeoAnalytics foca na análise de grandes quantidades de dados com ênfase em padrões espaciais e temporais.

Summarize Data

The Summarize Data toolset contains tools that calculate total counts, lengths, areas, and basic descriptive statistics of features and their attributes within areas or near other features.

  • Aggregate Points calculates statistics about points that fall within specified areas or bins.
  • Build Multi-Variable Grid generates a grid of square or hexagonal bins and calculates values for each bin based on the proximity of each input layer.
  • Describe Dataset calculates a summary of your input, and can optionally generate a sample of your input layer.
  • Join Features calculates statistics about features that share a spatial, temporal, or attribute relationship with other features.
  • Reconstruct Tracks calculates statistics about points or polygons that belong to the same track and reconstructs inputs into tracks.
  • Summarize Attributes calculates statistics about feature or tabular data that share attributes.
  • Summarize Within calculates statistics for area features and attributes that overlap each other.

Agregar Pontos

Utilize uma camada de feições de ponto ou uma camada de feições de área ou bandejas definidas por uma distância especificada, esta ferramenta determina quais pontos caem dentro de cada área ou bandeja e calcula estatísticas sobre todos os pontos dentro de cada área ou bandeja. Você pode opcionalmente aplicar um corte de tempo com esta ferramenta.

Os seguintes são exemplos:

  • Determinados os locais de ponto dos incidentes de crime, contar o número de crimes por município ou outro distrito administrativo.
  • Localizar as rendas mensais mais altas e mais baixas para locais de franquia utilizando caixas de 100-quilômetros .


Construir Grade de Múltiplas Variáveis

A ferramenta Construir Grade de Múltiplas Variáveis gera uma grade de caixas quadradas ou hexagonais e calcula variáveis para cada caixa baseado na proximidade de uma ou mais camadas de entrada.

Os seguintes são exemplos:

  • Com múltiplas camadas da infraestrutura de transporte público fornecidas, qual parte da cidade é menos acessível por transporte público?
  • Com camadas dos cursos de água fornecidas, como lagos e rios, qual é o nome do corpo de água mais próximo de cada local nos EUA?
  • Com uma camada de renda familiar fornecida, onde nos EUA a variação de renda nas 50 milhas circundantes é a maior?


Descrever Conjunto de Dados

Resuma as feições de entrada com estatísticas, camadas de amostra e visualização. Você pode optar por gerar uma camada de amostra ou camada de extensão.

Os seguintes são exemplos:

  • Consulte as estatísticas de resumo calculadas para entender os valores contidos em seu conjunto de dados.
  • Em vez de desenhar todo o conjunto de dados, crie uma camada de subconjunto para visualizar as feições e seus atributos quando adicionadas a um mapa.
  • Economize tempo e recursos e execute análises em uma camada de amostra antes de executar a análise em todo o conjunto de dados.


Ligar Feições

Com a utilização das camadas de feição ou dados tabelares, você pode ligar feições e registros baseados em relacionamentos específicos entre as camadas de entrada ou tabelas. As ligações serão determinadas por relacionamentos de atributos, espacial, temporal e a estatística sumária pode ser calculada opcionalmente.

Os seguintes são exemplos:

  • Dadas as localizações dos pontos de incidentes do crime com um tempo, junte os dados de crime para você mesmo, especificando um relacionamento espacial de crimes dentro de 1 quilômetro um do outro, e que ocorreu no prazo de uma hora um do outro, para determinar se há uma sequência de crimes perto um ao outro no espaço e tempo.
  • Com uma tabela de Ceps fornecida com informações demográficas e feições de área representando edifícios residenciais, ligue as informações demográficas com as residências de forma que cada residência agora tenha as informações.


Reconstruir Caminhos

Utilize uma camada habilitada ao tempo de feições de polígono e de ponto ou que representa um momento no tempo, esta ferramenta determina quais feições de entrada pertencem em um caminho e ordenará as entradas consecutivamente no tempo. As estatísticas são opcionalmente calculadas para as feições de entrada dentro de cada caminho.

Os seguintes são exemplos:

  • Dada as localizações de ponto e tempo das medições do furacão, calcule a velocidade média do vento e pressão máxima de vento do furacão.
  • Utilize as medições de GPS de navios para reconstruir rotas de navegação como linhas.


Resumir Atributos

Com a utilização de dados de feição ou tabelares, esta ferramenta resume as estatísticas dos campos especificados.

Os seguintes são exemplos:

  • Com as localizações de supermercados fornecidas com um campo COMPANY_NAME, resuma as lojas pelo nome da companhia para determinar estatísticas para cada companhia.
  • Com uma tabela de supermercados fornecida com campos COMPANY_NAME e COUNTY, resuma as lojas pelo nome da companhia e município para determinar estatísticas para cada companhia em cada município.


Resumir Dentro

Localiza áreas (e porções de áreas) que sobrepõem entre duas camadas e calcula a estatística da sobreposição.

Os seguintes são exemplos:

  • Dada uma camada de áreas de bacia e uma camada de áreas de uso da superfície por tipo de uso do solo, calcule a área total em acres do tipo de uso do solo para cada bacia.
  • Dada uma camada de lotes em um município e uma camada de limites da cidade, resuma o valor médio de lotes desocupados dentro de cada cidade.


Encontrar Locais

Estas ferramentas são utilizadas para identificar áreas que atendem vários critérios diferentes que você especifica.

  • Detectar Incidentes funciona com camadas habilitadas ao tempo para determinar quais feições atendem uma condição especificada.
  • Geocodificar Locais da Tabela converte endereços em coordenadas.
  • Encontrar Locais Semelhantes encontra locais mais semelhantes a um ou mais locais de referência baseado em critérios que você especifica.

Detectar Incidentes

Esta ferramenta funciona com uma camada de pontos, linhas, áreas ou tabelas habilitadas ao tempo que representa um instante no tempo. Utilizando feições ordenadas sequencialmente, denominadas trilhas, esta ferramenta determina quais feições são incidentes de interesse. Os incidentes são determinados pelas condições que você especifica.

Os seguintes são exemplos:

  • Detecta medições de furacão que são de Categoria 4 baseado em uma atributo de velocidade do vento.
  • Com a utilização de medições de GPS de movimento de navio com um campo representando a velocidade de navio, localize medições que são maiores que 1.5 vezes a média das cinco medições anteriores.


Geocodificar Locais da Tabela

Converte endereços em coordenadas. Utilize esta ferramenta em tabelas de compartilhamento do arquivo de grandes dados.


Encontrar Locais Semelhantes

Com base nos critérios especificados, encontre locais semelhantes ao medir a semelhança dos locais na sua camada de pesquisa do candidato para um ou mais locais de referência.

Os seguintes são exemplos:

  • Encontrar as dez lojas mais semelhantes através da análise do número de empregados e das vendas anuais.
  • Encontrar as 100 cidades mais semelhantes ao examinar a relação entre a população, crescimento anual e receitas fiscais.


Analisar Padrões

Estas ferramentas ajudam você a identificar, quantificar e visualizar modelos espaciais em seus dados.

  • Calcular Densidade obtém quantidades conhecidas de algum fenômeno e distribui estas quantidades pelo mapa.
  • Localizar Alto Valor de Incidência identifica agrupamentos estatisticamente significativos no modelo espacial dos seus dados.
  • Localizar Agrupamentos de Ponto localiza agrupamentos de feições de ponto ao redor de ruído baseado em sua distribuição espacial.
  • Classificação e Regressão baseadas em Floresta cria modelos e gera previsões utilizando uma adaptação do algoritmo de floresta aleatória de Leo Breiman, um método supervisionado de aprendizado automático.
  • Regressão Linear Generalizada gerar previsões ou modela uma variável dependente em termos de sua relação com um conjunto de variáveis explanatórias.

Calcular Densidade

A ferramenta Calcular Densidade cria um mapa de densidade a partir das feições de ponto, espalhando quantidades conhecidas de algum fenômeno (representado como atributos de pontos) através do mapa. O resultado é uma camada de áreas representando a densidade.

Os seguintes são exemplos:

  • Calcular densidades dos hospitais dentro de um município. A camada resultante mostrará áreas com acessibilidade alta e baixa para hospitais, e estas informações podem ser utilizadas para decidir onde novos hospitais devem ser construídos.
  • Identificar áreas que estão em alto risco de incêndio em florestas baseado em histórico dos locais de incêndio em florestas.
  • Localizar comunidades que estão longe de estradas importantes para planejar onde novas estradas devem ser construídas.


Localizar Agrupamentos de Pontos

A ferramenta Localizar Agrupamentos de Pontos localiza agrupamentos de feições de ponto ao redor de ruído baseado em sua distribuição espacial.

Os seguintes são exemplos:

  • Localize agrupamentos de-casas infestadas por pragas para ajudar nos esforços de erradicação de alvo.
  • Informe e atue no salvamento e necessidades de evacuação baseado no tamanho e localização dos agrupamentos utilizando tweets localizados geograficamente seguindo perigos naturais ou ataques.


Localizar Valor Alto de Incidência

A ferramenta Localizar Valor Alto de Incidência determinará se existe qualquer agrupamento estatisticamente significativo no modelo espacial dos seus dados.

  • Seus pontos estão (incidentes de crimes, árvores, acidentes de tráfego) realmente agrupados? Como você pode estar certo?
  • Você realmente descobriu um valor alto de incidência estatisticamente significativo (para gastos, mortalidade infantil, pontuações de teste constantemente altas) ou seu mapa contará uma história diferente se você alterou o modo que foi simbolizado?
A ferramenta Localizar Valor Alto de Incidência o ajudará a responder estas perguntas com confiança.


Classificação e Regressão baseada em Floresta

A ferramenta Classificação e Regressão baseadas em Floresta gera previsões utilizando uma adaptação do algoritmo de floresta aleatória de Leo Breiman, que é um método supervisionado de aprendizado automático.

Os seguintes são exemplos:

  • Considerando os dados sobre a ocorrência de ervas marinhas, como também, um número de variáveis ambientais explanatórias representadas como ambos os atributos que foram enriquecidos utilizando uma grade de múltiplas variáveis para calcular as distâncias para as fábricas a montante e principais portos, futura ocorrência de ervas marinhas pode ser prevista com base em projeções futuras para estas mesmas variáveis explanatórias ambientais.
  • Suponha que você tenha dados de rendimento de culturas em centenas de fazendas em todo o país junto com outros atributos em cada uma destes fazendas (número de funcionários, área plantada e assim por diante). Utilizando estes dados, você pode fornecer um conjunto de feições que representam fazendas onde você não tem rendimento de safra (mas tem todas as outras variáveis) e fazer uma previsão sobre o rendimento da safra.
  • Os valores da habitação podem ser previstos com base nos preços das casas que foram vendidas no ano em curso. O preço de venda das casas vendidas, juntamente com informações sobre o número de quartos, a distância das escolas, a proximidade das principais rodovias, a renda média e a contagem de crimes podem ser utilizados para prever preços de venda de casas semelhantes.


Regressão Linear Generalizada

A ferramenta Regressão Linear Generalizada gera previsões ou modela uma variável dependente em termos de sua relação com um conjunto de variáveis explanatórias. Esta ferramenta pode ser utilizada para ajustar os modelos contínuos (OLS), binário (logística) e contagem (Poisson).

Os seguintes são exemplos:

  • Quais características demográficas contribuem para as altas taxas de uso do transporte público?
  • Existe uma relação positiva entre vandalismo e roubo?
  • Quais variáveis efetivamente prevêem o volume de chamadas 911? Considerando as projeções futuras, qual é a demanda esperada pelos recursos de resposta a emergências?
  • Quais variáveis afetam as baixas taxas de natalidade?


Utilizar Proximidade

Estas ferramentas ajudam a responder uma das perguntas mais comuns apresentada na análise espacial: O que está próximo do que?

  • Criar Buffers cria áreas de uma distância especificada de feições.

Criar Buffers

O buffer é uma área que cobre uma determinada distância a partir de uma feição de polígono, linha ou ponto.

Os seguintes são exemplos:

  • Utilizar feições de rio lineares, criar buffer de cada rio em 50 vezes a largura do rio para determinar um limite ciliar proposto.
  • Dadas as áreas representando países, crie um buffer para cada país em 200 milhas náuticas para determinar o limite marítimo.


Manage Data

These tools are used for the day-to-day management of geographic and tabular data.

  • Append Data appends features to an existing hosted feature layer.
  • Calculate Field creates a new field or modifies an existing field and creates a new layer in your contents.
  • Clip Layer copies features that intersect a defined area of interest.
  • Copy to Data Store copies data to your ArcGIS Data Store and creates a layer in your contents.
  • Dissolve Boundaries merges areas that share a common boundary or attribute value.
  • Merge Layers copies all features from two layers into a single result layer.
  • Overlay Layers combines two or more layers into a single layer.

Copiar para Armazenamento de Dados

Copia uma camada de feição de entrada ou tabela para um ArcGIS Data Store e cria uma camada em seu ArcGIS Enterprise.

Os seguintes são exemplos:

  • Copiar uma coleção de .csv arquivos CSV em um compartilhamento de um grande arquivo de dados para armazenamento de dados de espaço-tempo para visualização.
  • Copiar as feições na extensão de mapa atual que são guardadas no armazenamento de dados de espaço-tempo no armazenamento de dados relacional.


Calcular Campo

Calcule valores para um campo novo ou existente e crie uma camada em seu conteúdo em ArcGIS Enterprise.

Os seguintes são exemplos:

  • Modificar um campo existente denominado total para ser a soma das receitas dos campos total_2016, total_2017, e total_2018.
  • Criar um novo campo para categorizar os níveis de perigo com base em valores de campo, como windspeed e pollutant.


Camada de Recorte

Recorta as feições de entrada das áreas de interesse definidas. O resultado da saída será um subconjunto de feições de entrada.

Os seguintes são exemplos:

  • Cria um subconjunto de edifícios em risco que se enquadrem nas áreas da zona de inundação.
  • Determina quais as principais estradas que percorrem suas áreas potenciais de desenvolvimento.


Dissolver Limites

Junta as feições de área que se interseccionam ou compartilham um valor de campo em comum para criar feições de múltiplas partes ou contíguas.

Os seguintes são exemplos:

  • Cria uma única feição representando o estado do Havaí, dissolvendo os limites das ilhas havaianas.
  • Combina áreas de estudo semelhantes com base no tipo de solo para produzir um conjunto de dados de tipo de solo contíguo.


Sobrepor Camadas

Combina duas ou mais camadas em uma única camada. A sobreposição é utilizada para responder uma das perguntas mais básicas da geografia,O que está por cima do que?

O seguinte são exemplos:

  • Quais lotes estão dentro da área de inundação de 100 anos? (Dentro é somente outro modo de dizer sobre.)
  • Qual uso do solo está sobre qual tipo de solo?
  • Quais poços estão dentro de bases militares abandonadas?


Acrescentar Dados

Anexa feições para umas camada hospedada existentes em seu conteúdo em ArcGIS Enterprise.

Os seguintes são exemplos:

  • Atualize sua camada de feição 2018_Sales cada mês que seus relatórios forem disponibilizados.
  • Anexe conjunto de dados de acompanhamento de furacão a partir dos anos 2016 e 2017 em um conjunto de dados de furacão mestre.
  • Colete e anexe conjunto de dados de várias origens com esquemas diferentes utilizando mapeamento de campo personalizado.


Juntar Camadas

Combina duas camadas de feição para criar uma única camada resultante. Todos os campos da camada de junção serão incluídos por padrão ou você pode especificar regras de junção personalizadas para definir o esquema resultante.

Os seguintes são exemplos:

  • Junta três camadas contendo informações demográficas da Inglaterra, País de Gales e Escócia para criar uma única camada da Grã-Bretanha.
  • Combina duas camadas contendo informações de lote para municípios contíguos, mantendo somente os campos que têm o mesmo nome e tipo nas duas camadas.


Enriquecimento de Dados

Estas ferramentas ajudam você a explorar o caráter das áreas.

  • Enriquecer de Grade de Múltiplas Variáveis associa atributos de uma camada de grade de múltiplas variáveis a uma camada de ponto.

Enriquecer de Grade de Múltiplas Variáveis

Une eficientemente os atributos de uma grade de múltiplas variáveis a uma camada de ponto, permitindo que você adicione rapidamente uma grande e diversificada coleção de informações para apontar dados para uso em análises espaciais posteriores.

Os seguintes são exemplos:

  • Considerando uma camada contendo milhões de incidentes de interrupção de energia, aprimore as feições do incidente com informações sobre o uso típico, os riscos ambientais e as condições de infraestrutura, a fim de estudar a relação entre estes fatores e a frequência da falta de energia.
