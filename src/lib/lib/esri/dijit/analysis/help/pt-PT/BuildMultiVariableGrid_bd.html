Construi Grelha Multivariável

A ferramenta Construir Grelha Multivariável funciona com uma ou mais camadas de elementos de ponto, linha ou polígono. A ferramenta gera uma grelha de caixas quadradas ou hexagonais e calcula valores para cada caixa com base na proximidade de cada camada de entrada. Estes valores podem incluir as seguintes variáveis:

  • Distância para Mais Próximo—A distância entre cada caixa e o elemento mais próximo
  • Atributo de Mais Próximo—Um valor de campo do elemento mais próximo de cada caixa
  • Resumo de Proximidade—Uma estatística especificada de todos os elementos a determinada distância de cada caixa
  • Resumo de Interseções—Uma estatística especificada de todos os elementos que intersetam cada caixa

Apenas variáveis que especificar serão incluídas na camada resultante. Estas variáveis podem ajudá-lo a compreender a proximidade e distribuição dos seus dados. Os resultados podem ajudá-lo a começar a responder a questões tais como as seguintes:

  • Tendo em conta várias camadas de infraestruturas de transportes públicos, que parte da cidade é menos acessível com recurso a transportes públicos?
  • Tendo em conta camadas de vias aquáticas, tais como lagos e rios, qual é o nome do corpo aquático mais próximo de cada localização nos E.U.A.?
  • Tendo em conta a camada de rendimento familiar, em que os E.U.A. representam a variação de rendimento, em que parte dos E.U.A. se encontra a maior variação de rendimento nas 50 milhas circundantes?

Selecione o tipo de caixas que serão utilizadas para construir a grelha multivariável

O tipo e o tamanho das caixas que serão utilizadas para criar a grelha multivariável Para a opção Quadrado , o número e unidades especificados determinam a altura e o comprimento do quadrado. Para a opção Hexágono , o número e unidades especificados determinam a distância entre lados paralelos.

Esta análise utiliza caixas e requer um sistema de coordenadas projetado. Pode definir o Sistema de coordenadas de processamento nos Ambientes de Análise. Se o seu sistema de coordenadas de processamento não se encontra definido para um sistema de coordenadas projetado, será instado a defini-lo quando Correr Análise .

Selecione camadas de entrada e variáveis que pretende incluir na grelha

Clique no botão adicionar ( +) para abrir a janela Adicionar Variáveis e adicionar uma camada à sua análise. Na janela Adicionar Variáveis , selecione uma camada de entrada e configure as variáveis que pretende calcular para essa camada de entrada.

Após ter escolhido e configurado as variáveis que pretende calcular, a camada será listada na janela da ferramenta com quaisquer outras camadas selecionadas. Clique no botão editar em qualquer camada para abrir a janela Adicionar Variáveis e visualizar todas as variáveis configuradas para essa camada. Clique no botão eliminar em qualquer camada para remover essa camada e todas as variáveis relacionadas da análise.

Adicionar variáveis

Esta janela permite-lhe selecionar uma camada de entrada e configurar as variáveis que pretende calcular para essa camada de entrada. Para começar, utilize o menu pendente Selecionar uma camada de entrada para selecionar a camada de ponto, polígono ou linha que pretende adicionar à sua análise.

Para além de escolher uma camada do seu mapa, pode escolher Escolher Camada de Análise na parte inferior da lista pendente para navegar até aos seus conteúdos de um conjunto de dados de partilha de ficheiros de big data ou camada de elementos.

Após ter selecionado uma camada de entrada, clique no botão Adicionar uma variável para abrir uma nova variável para edição. Tem de atribuir um nome único a cada variável. Este será o nome do campo que é criado na tabela de atributos da camada resultante.

Utilize o menu pendente para selecionar o tipo de variável a adicionar. Cada tipo de variável tem de ser configurado com um conjunto de parâmetros único, e todos os parâmetros são obrigatórios.

Após ter adicionado todas as variáveis pretendidas para a camada selecionada, clique no botão Adicionar para guardar as variáveis.

Nome da camada resultante

O nome da camada que será criada. Se estiver a escrever para ArcGIS Data Store, os seus resultados serão gravados em O Meu Conteúdo e adicionados ao mapa. Se estiver a escrever para uma partilha de ficheiros big data, os seus resultados serão guardados numa partilha de ficheiros big data e adicionados ao seu manifesto. Não serão adicionados ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Caso a camada já existe, a ferramenta irá falhar.

Ao escrever para ArcGIS Data Store (armazenamento relacional ou espaciotemporal de dados big data) utilizando a caixa suspensa Guardar resultado em pode especificar o nome de uma pasta em O Meu Conteúdo, onde será armazenado o resultado.
