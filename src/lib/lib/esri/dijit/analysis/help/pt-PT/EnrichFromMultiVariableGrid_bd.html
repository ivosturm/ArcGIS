Melhorar a Partir de Uma Grelha de Múltiplas Variáveis

A ferramenta Melhorar a Partir de Uma Grelha de Múltiplas Variáveis junta eficientemente atributos de uma grelha de múltiplas variáveis a uma camada de pontos.

Uma grelha de múltiplas variáveis é qualquer camada criada com a ferramenta Construir Grelha de Múltiplas Variáveis, que compila atributos de múltiplas camadas para uma única grelha de polígonos. Utilizar uma grelha de múltiplas variáveis para melhoria permite-lhe adicionar rapidamente grandes e diversas recolhas de informação a dados de pontos para utilização em posteriores análises espaciais.

Por exemplo, suponhamos que obteve uma camada de novos eventos de crime que ocorreram no último ano numa cidade. Pretende modelar as relações entre a frequência dos crimes e uma diversidade de fatores demográficos, económicos e geográficos. Tem acesso a dados que descrevem estes fatores, mas estão armazenados em cinco diferentes camadas com distintos tipos de geometria. Além disso, algumas camadas estão armazenadas numa partilha de ficheiros de big data enquanto outras estão armazenadas como serviços de elementos. Criando uma grelha de múltiplas variáveis para armazenar esta informação e, em seguida, juntando a grelha aos seus eventos de crime com a ferramenta Melhorar a Partir de Uma Grelha de Múltiplas Variáveis, poderá juntar eficientemente uma ampla variedade de atributos à sua camada de pontos com apenas duas execuções da ferramenta. Pode depois utilizar os atributos que se juntaram como variáveis explanatórias e começar a criar um modelo de frequência dos crimes com a camada resultante.

Escolher os elementos de pontos a melhorar

Os elementos de pontos que serão melhorados pela grelha de múltiplas variáveis.

Para além de escolher uma camada do seu mapa, pode escolher Escolher Camada de Análise na parte inferior da lista pendente para navegar até aos seus conteúdos de um conjunto de dados de partilha de ficheiros de big data ou camada de elementos.

Escolher a grelha de múltiplas variáveis a partir da qual melhorar

A camada de grelha de múltiplas variáveis que contém atributos que se irão juntar aos elementos de pontos de entrada. A camada terá de ter sido criada utilizando a ferramenta Construir Grelha de Múltiplas Variáveis e tem de ser uma camada de elementos alojada. Camadas de grelha de múltiplas variáveis armazenadas numa partilha de ficheiros de big data não são suportadas como entrada para esta ferramenta.

Escolher as variáveis a adicionar

Os atributos na camada de grelha de múltiplas variáveis que se irão juntar aos elementos de pontos de entrada sempre que os elementos de pontos intersetem a grelha.

Nome da camada resultante

O nome da camada que será criada. Se estiver a escrever para ArcGIS Data Store, os seus resultados serão gravados em O Meu Conteúdo e adicionados ao mapa. Se estiver a escrever para uma partilha de ficheiros big data, os seus resultados serão guardados numa partilha de ficheiros big data e adicionados ao seu manifesto. Não serão adicionados ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Caso a camada já existe, a ferramenta irá falhar.
