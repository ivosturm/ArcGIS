Построить сеть по нескольким переменным

Инструмент Построить сеть по нескольким переменным работает с одним или несколькими слоями точечных, линейных или полигональных объектов. Инструмент создает сетку квадратных или шестиугольных ячеек и вычисляет значения в каждой из них на основании близости входных слоев. Эти изменения могут включать следующие переменные:

  • Расстояние до ближайшего—расстояние от каждой ячейки до ближайшего объекта
  • Атрибут ближайшего—значение атрибутивного поля объекта, который находится ближе всего к ячейке
  • Сумма ближайших—статистика по всем объектам в пределах заданного расстояния от каждой ячейки
  • Статистика по пересечению—статистика по всем объектам, которые пересекают ячейку

В итоговый слой попадут только те переменные, которые вы указали. Эти переменные могут помочь понять близость и распределение ваших данных. Результаты помогут найти ответы на такие вопросы, как:

  • Как на основе данных об инфраструктуре общественного транспорта понять, какая часть города является наименее доступной?
  • Как на основе данных о водных объектах (реках и озерах) узнать, какой из них является ближайшим к каждой точке на территории США?
  • Как на основе данных о доходах понять, где в США наибольшие различия по этому показателю в пределах 50 миль?

Выбор типа ячеек, которые будут использоваться для построения сети

Тип и размер ячеек, которые будут использоваться для построения сети на основе нескольких переменных. Если выбрана опция Квадрат, то необходимо задать величину и единицы измерения, которые будут определять длину и высоту стороны квадратной ячейки. Для Шестиугольника число и единицы измерения определяют расстояние между параллельными сторонами.

Этот анализ использует бины и требует системы координат проекции. Вы можете настроить Систему координат обработки в Параметрах среды анализа. Если система координат обработки не является системой координат проекции, при Запуске анализа вам предложат изменить её. .

Выбор входных слоев и переменных, которые будут включены в сеть

Нажмите кнопку добавления ( +), чтобы открыть окно Добавить переменные и укажите слои для анализа. В окне Добавить переменные выберите входной слой и настройте переменные, которые хотите вычислить для входного слоя.

После выбора и настройки переменных слой появится в окне инструмента наряду с остальными выбранными слоями. Нажмите кнопку редактирования для любого слоя, чтобы открыть окно Добавить переменные и просмотрите все переменные, настроенные для этого слоя. Нажмите кнопку удаления для любого слоя, чтобы удалить его и все связанные с ним переменные из анализа.

Добавление переменных

Это окно позволяет выбрать входной слой и настроить переменные, которые необходимо для него вычислить. Для начала воспользуйтесь меню Выбрать входной слой, чтобы указать точечный, полигональный или линейный слой, который вы хотите использовать в ходе анализа.

Кроме выбора слоя из вашей карты, можно щелкнуть Выбрать слой анализа в нижней части ниспадающего списка, чтобы найти свои ресурсы для набора данных или векторного слоя файлового хранилища больших данных.

Затем нажмите кнопку Добавить переменную, чтобы открыть новую переменную на редактирование. Для каждой переменной необходимо задать уникальное имя. Это будет имя поля, которое будет создано в атрибутивной таблице итогового слоя.

Используйте ниспадающее меню для выбора типа добавляемой переменной. Каждый тип переменной должен быть настроен с уникальным набором параметров, причем параметры нужно указать все.

После добавления всех необходимых переменных к выбранном слою, нажмите кнопку Добавить, чтобы сохранить переменные.

Имя слоя результата

Имя создаваемого слоя. Если вы записываете в ArcGIS Data Store, ваши результаты будут сохранены в Моих ресурсах и добавлены на карту. Если вы записываете в файловое хранилище больших данных, ваши результаты будут сохранятся в файловом хранилище больших данных и добавляться в его файл манифеста. Они не будут добавлены на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если слой уже существует, произойдет сбой.

При записи в ArcGIS Data Store (реляционное или пространственно-временное хранилище больших данных) с помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
