Копировать в хранилище данных

Инструмент копирует входной слой в хранилище данных. Вы можете копировать результаты в ArcGIS Data Store (реляционное хранилище или хранилище больших пространственно-временных данных) с созданием слоя на портале, или копировать результаты в файловое хранилище больших данных.

Например, вы можете скопировать объекты, хранящиеся в файловом хранилище больших данных в реляционное хранилище, при этом скопировав только объекты в пределах текущего экстента карты. При этом будет создан размещенный сервис объектов, содержащий объекты, которые находились в пределах текущего экстента карты.

Выбрать слой для копирования

Копируемый слой или таблица.

Кроме выбора слоя из вашей карты, можно щелкнуть Выбрать слой анализа в нижней части ниспадающего списка, чтобы найти свои ресурсы для набора данных или векторного слоя файлового хранилища больших данных.

Для выборки можно использовать все объекты (дополнительно)

Дополнительно, используйте кнопку выборки для идентификации копируемых местоположений.

Выберите целевое хранилище данных

Хранилище данных, в которое будет скопирован слой. Этот параметр имеет приоритет над заданным в Параметрах среды анализа. Используются следующие опции:

  • ArcGIS Data Store ArcGIS Data Store
    • Пространственно-временное – используется по умолчанию.
    • Реляционное
  • Шаблоны выходного файлового хранилища больших данных
    • Список доступных шаблонов файлового хранилища больших данных в формате bigDataFileShareName-templateName. Шаблоны файлового хранилища больших данных должны быть настроены перед запуском инструмента.

Имя слоя результата

Имя создаваемого слоя. Если вы записываете в ArcGIS Data Store, ваши результаты будут сохранены в Моих ресурсах и добавлены на карту. Если вы записываете в файловое хранилище больших данных, ваши результаты будут сохранятся в файловом хранилище больших данных и добавляться в его файл манифеста. Они не будут добавлены на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если слой уже существует, произойдет сбой.

При записи в ArcGIS Data Store (реляционное или пространственно-временное хранилище больших данных) с помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
