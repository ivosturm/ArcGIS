Обогатить из Сетки нескольких переменных

Инструмент Обогатить из Сетки нескольких переменных эффективно присоединяет атрибуты из сетки нескольких переменных к точечному слою.

Сетка нескольких переменных - это любой слой, созданный с помощью инструмента Построить сетку нескольких переменных, который компилирует атрибуты из нескольких слоев в одну сетку полигонов. Использование сетки нескольких переменных для обогащения данных позволяет быстро добавлять к точечным данным большие и разнообразные коллекции информации для использования в последующем анализа.

Например, предположим вы получили слой с данными о преступлениях в городе за прошлый год. Вы хотите создать модель взаимосвязей между частотой преступлений и разнообразными демографическими, экономическими и географическими факторами. У вас есть доступ к данным, описывающим эти факторы, но они хранятся в пяти различных слоях с различным типом геометрии. Кроме того, некоторые слои хранятся в файловомхранилищебольшихданных, а другие - в виде сервисов объектов. Создав сетку нескольких переменных для хранения этой информации и затем объединив сетку с событиями преступлений с помощью инструмента Обогатить из Сетки нескольких переменных, вы сможете эффективно присоединить множество атрибутов к точечному слою, использовав всего два запуска инструментов. Затем вы можете использовать объединенные атрибуты как описательные переменные, чтобы начать строить модель частоты преступлений.

Выберите точечный слой для обогащения

Точечный слой, который будет обогащен данными из сетки нескольких переменных.

Кроме выбора слоя из вашей карты, можно щелкнуть Выбрать слой анализа в нижней части ниспадающего списка, чтобы найти свои ресурсы для набора данных или векторного слоя файлового хранилища больших данных.

Выберите сетку нескольких переменных для обогащения

Слой сетки нескольких переменных, содержащий атрибуты, которые будут присоединены к входным точечным объектам. Это размещенный векторный слой, который должен быть создан инструментом Построить сетку нескольких переменных. Слои сетки нескольких переменных, хранящиеся в файловом хранилище больших данных, не поддерживаются этим инструментом.

Выберите переменные для добавления

Атрибуты из слоя сетки нескольких переменных, которые будут присоединяться ко входным точечным объектам при каждом пересечении сетки объектами.

Имя слоя результата

Имя создаваемого слоя. Если вы записываете в ArcGIS Data Store, ваши результаты будут сохранены в Моих ресурсах и добавлены на карту. Если вы записываете в файловое хранилище больших данных, ваши результаты будут сохранятся в файловом хранилище больших данных и добавляться в его файл манифеста. Они не будут добавлены на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если слой уже существует, произойдет сбой.
