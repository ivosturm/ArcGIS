Поиск кластеров точек

Инструмент Поиск кластеров точек определяет среди точечных объектов кластеры в окружающем шуме на основе их пространственного распределения.

Например, неправительственная организация изучает частный случай распространения заболевания. Есть точечный набор данных, представляющий домохозяйства на изучаемой территории, некоторые из которых заражены, а некоторые - нет. С помощью инструмента Поиск кластеров точек специалист может определить кластеры зараженных домохозяйства, это поможет выявить области, где надо начинать лечение и борьбу с переносчиками.

Выберите слой, для которого будут найдены кластеры

Точечный слой, в котором нужно выявить кластеры. У слоев должна быть задана пространственная привязка или пространственная привязка обработки должна быть задана как система координат проекции с помощью Параметров среды анализа.

Кроме выбора слоя из вашей карты, можно щелкнуть Выбрать слой анализа в нижней части ниспадающего списка, чтобы найти свои ресурсы для набора данных или векторного слоя файлового хранилища больших данных.

Выберите используемый метод кластеризации

Метод кластеризации, который используется для отделения кластеров точечных объектов от окружающего шума. Вы можете выбрать заданное расстояние или самонастраивающийся механизм кластеризации.

Заданное расстояние (DBSCAN) использует указанный радиус поиска для отделения плотных кластеров от окружающего шума. Заданное расстояние (DBSCAN) является наиболее быстрым методом кластеризации, но подходит только при наличии очень четкого расстояния поиска, что хорошо подходит для поиска всех представленных кластеров. Заданное расстояние (DBSCAN) находит кластеры с похожей плотностью.

Самонастраивающаяся кластеризация (HDBSCAN) не требует задания диапазона поиска, но является более длительным методом. Самонастраивающаяся кластеризация (HDBSCAN) находит кластеры точек сходным с Заданное расстояние (DBSCAN) методом, но использует варьирующие диапазоны поиска, что допускает наличие кластеров с разной плотностью.

Минимальное число точек, которые будут считаться кластером

Этот параметр используется по-разному, в зависимости от выбранного метода кластеризации:

  • Заданное расстояние (DBSCAN) - задает число объектов, которое должно быть найдено в пределах указанного расстояния от точки, с которой начинается формирование кластера. Расстояние задается параметром Ограничьте диапазон поиска до.
  • Самонастраивающаяся кластеризация (HDBSCAN) - задает число объектов, окружающих каждую точку (включая саму точку), которые будут учитываться при оценке плотности. Это число также соответствует минимальному размеру кластера, размешенному при извлечении кластеров.

Ограничить диапазон поиска до

При использовании с методом Заданное расстояние (DBSCAN), этот параметр является расстоянием, в пределах которого должно быть найдено Минимальное число точек, которые будут считаться кластером. Этот параметр не используется при выборе метода Самонастраивающаяся кластеризация (HDBSCAN).

Имя слоя результата

Имя создаваемого слоя. Если вы записываете в ArcGIS Data Store, ваши результаты будут сохранены в Моих ресурсах и добавлены на карту. Если вы записываете в файловое хранилище больших данных, ваши результаты будут сохранятся в файловом хранилище больших данных и добавляться в его файл манифеста. Они не будут добавлены на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если слой уже существует, произойдет сбой.

При записи в ArcGIS Data Store (реляционное или пространственно-временное хранилище больших данных) с помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
