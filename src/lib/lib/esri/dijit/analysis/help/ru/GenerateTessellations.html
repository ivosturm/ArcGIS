Создать замощение

Инструмент создает замощения на исследуемой области на основе заданных формы и размера.

Опция Использовать текущий экстент карты отключена для этого инструмента, так как экстент, используемый для создания замощения, управляется параметром Указать экстент.

Выбрать геометрию

Форма или геометрия создаваемого замощения. Доступны формы: Квадрат, Шестиугольник, Поперечный шестиугольник, Треугольник или Ромб.

Указать область изучения

Экстент или область, которая покрывается замощением. Экстент можно задать с помощью текущего экстента карты, экстента существующего слоя или путем интерактивного рисования области, которая будет использоваться в качестве экстента.

Если для определения экстента был выбран векторный слой, будет включена опция для сохранения только пересекающегося замощения. Если выбрано, то в результирующем слое замощения будет создано замощение, которое покрывает или пересекается с объектами в указанном слое экстента.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
