คัดลอกไปยังแหล่งเก็บข้อมูล

เครื่องมือนี้จะให้ใส่ชั้นข้อมูลอินพุธและคัดลอกไปยัง data store คุณสามารถคัดลอกผลลัพธ์ไปยัง ArcGIS Data Store (ที่จัดเก็บข้อมูลขนาดใหญ่เชิงสัมพันธ์หรือเชิงพื้นที่ชั่วคราว) และชั้นข้อมูลจะถูกสร้างในพอร์ทัลของคุณ หรือคุณสามารถคัดลอกผลลัพธ์ไปยังไฟล์ข้อมูลขนาดใหญ่

ตัวอย่างเช่น คุณสามารถคัดลอกฟีเจอร์ที่ถูกเก็บไว้ใน ข้อมูลขนาดใหญ่ใช้ไฟล์ร่วมกัน ไปยัง เก็บข้อมูลเชิงสัมพันธ์ และระบุว่ามีเพียงฟีเจอร์ในขอบเขตแผนที่ปัจจุบันเท่านั้นจะถูกคัดลอก สร้างฟีเจอร์เซอร์วิสหลักนี้กับฟีเจอร์เหล่านั้นที่อยู่ในขอบเขตแผนที่ที่กำหนด

เลือกชั้นที่จะคัดลอก

ชั้นข้อมูลหรือตารานั้นที่จะคัดลอก

นอกเหนือจากการเลือกชั้นข้อมูลจากแผนที่คุณสามารถเลือก เลือกชั้นข้อมูลการวิเคราะห์ ที่ด้านล่างของรายการแบบเลื่อนลงเพื่อเรียกดูเนื้อหาของคุณสำหรับชุดข้อมูลขนาดใหญ่หรือชั้นฟีเจอร์

คุณอาจจะใช้ฟีเจอร์ทั้งหมดหรือเลือกบางส่วน (ตัวเลือก)

หรือใช้ปุ่มเลือกเพื่อระบุตำแหน่งที่จะคัดลอก

เลือก ที่จัดเก็บ ข้อมูลที่จะคัดลอก

Data Store เป็นการเก็บชั้นข้อมูลที่จะถูกคัดลอกไป พารามิเตอร์นี้จะลบล้างการเลือกใน สภาพแวดล้อมการวิเคราะห์ ตัวเลือกที่มีดังต่อไปนี้:

  • ArcGIS Data Store ArcGIS Data Store
    • พื้นที่ชั่วคราว—นี้คือค่าตั้งต้น
    • เชิงสัมพันธ์
  • แม่แบบผลลัพธ์การแชร์ไฟล์ข้อมูลขนาดใหญ่
    • รายการของแม่แบบการแชร์ไฟล์ข้อมูลขนาดใหญ่ในรูปแบบ bigDataFileShareName-templateName. ต้องกำหนดค่าแม่แบบการแชร์ไฟล์ข้อมูลขนาดใหญ่ก่อนเรียกใช้งานเครื่องมือ

ชื่อของชั้นข้อมูลผลลัพธ์

ชื่อของชั้นข้อมูลที่จะถูกสร้างขึ้น หากคุณกำลังเขียนถึง ArcGIS Data Store, ผลลัพธ์ ของคุณจะถูกบันทึกเก็บในMy Content และเพิ่มเข้าในแผนที่ หากคุณกำลังเขียนไปยังไฟล์แชร์ข้อมูลขนาดใหญ่, ผลลัพธ์ ของคุณจะถูกเก็บในไฟล์แชร์ข้อมูลใหญ่นั้น และเพิ่้มใน ข้อความ ที่แสดง ซึ่งจะ ไม่ถูกเพิ่มเข้าในแผนที่ ชื่อเริ่มต้นจะขึ้นอยู่กับชื่อเครื่องมือและชื่อชั้นข้อมูลอินพุต ถ้าชั้นข้อมูลมีอยู่แล้วเครื่องมือจะล้มเหลว

เมื่อคุณเขียน ArcGIS Data Store (แหล่งข้อมูลใหญ่สเปชิโอเทมโพรัลหรือเชิงสัมพันธ์) ที่ใช้เพื่อ บันทึกผลลัพธ์ในช่องแบบเลื่อนลง , คุณสามารถระบุชื่อของโฟลเดอร์ใน My Content ซึ่งจะถูกบันทึกไว้ในโฟลเดอร์นี้
