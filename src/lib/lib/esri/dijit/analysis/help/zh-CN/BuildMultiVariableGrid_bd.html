构建多变量格网

“构建多变量格网”工具适用于一个或多个点、线或面要素图层。该工具会生成方形和六边形条柱格网，并根据每个输入图层的邻近性计算每个条柱的值。这些值可包括以下变量：

  • 与最近要素的距离 - 每个条柱与最近要素的距离
  • 最近要素属性 - 距离每个条柱最近的要素的字段值
  • 邻近要素汇总 - 距离每个条柱的某一范围内所有要素的指定统计数据
  • 相交要素汇总 - 与每个条柱相交的所有要素的指定统计数据

仅在结果图层中包含所指定的变量。这些变量可帮助您了解数据的邻近性和分布。结果可以帮助您开始回答以下问题：

  • 已知多个公共交通基础设施图层，城市的哪个区域公共交通最不便利？
  • 已知水体（例如湖泊和河流）图层，最接近美国各区域的水体名称是什么？
  • 已知家庭收入图层，美国哪个位置周围 50 英里范围内收入差距最大？

选择将用于构建多变量格网的条柱类型

用于创建多变量格网的条柱类型和大小。对于 方形选项，由指定的数字和单位决定正方形的高度和长度。对于 六边形选项，由指定的数字和单位决定平行边之间的距离。

该分析使用图格，并且需要投影坐标系。您可以在 分析环境中设置 处理坐标系。如果您的处理坐标系未设置为投影坐标系，则 运行分析时系统将提示您进行设置 。

选择要包含在格网中的输入图层和变量

单击添加按钮 ( +) 打开 添加变量窗口，然后向分析添加图层。在 添加变量窗口中，选择输入图层，然后为该输入图层配置要计算的变量。

选择并配置要计算的变量后，该图层将与任何其他所选图层一起在工具窗口中列出。针对任何图层单击“编辑”按钮可打开 添加变量窗口并查看已为该图层配置的所有变量。针对任何图层单击“删除”按钮可从分析中删除该图层及所有相关变量。

添加变量

可在此窗口中选择输入图层并为该输入图层配置要计算的变量。首先，在 选择输入图层下拉菜单中选择要添加到分析中的点、面或线图层。

除了从地图中选择图层外，还可以在下拉列表的底部选择 选择分析图层，以浏览至大数据文件共享数据集或要素图层中的内容。

选择输入图层后，单击 添加变量按钮打开一个新变量以供编辑。必须为每个变量提供唯一名称。该名称将是在结果图层的属性表中创建的字段名称。

使用下拉菜单选择要添加的变量类型。必须使用唯一的参数集配置每个变量类型，并且所有参数都是必填参数。

为所选图层添加所有所需变量后，请单击 添加按钮保存变量。

结果图层名称

要创建的图层的名称。如果您要写入 ArcGIS Data Store，则您的结果将保存在我的内容并添加到地图中。如果您要写入大数据文件共享，则您的结果将存储于大数据文件共享中，并添加到其清单。结果将不会添加到地图中。默认名称基于工具名称以及输入图层名称。如果该图层已存在，将无法使用该工具。

如果写入 ArcGIS Data Store（关系或时空大数据存储），可以使用将结果保存在下拉框指定 我的内容中的文件夹的名称，结果将保存到该文件夹中。
